turtle

海龟绘图，适合用于引导孩子学习编程。

控制箭头

画笔移动距离和角度 forward，backward，left，right
画笔到指定坐标 setx ，sety，setpos，goto，setheading，home
画圆 circle(半径，幅度，步数)
画点 dot(直径，颜色)
标记箭头 stamp，clearstamp(id),cleanstamps(n)，删除前后 n 个标记
取消 undo
speed，设置速度 0 最快，1-10 逐渐快

获取状态

position,pos,distance, towards 返回海归位置到初始位置连线夹角

画笔控制

pendown，penup，pensize（线宽度）
pen，返回或者设置画笔属性
isdown，检测笔是否落下
pencolor 设置或返回画笔颜色

颜色填充

begin_fill，end_fill ,填充
hideturtle ,showturtle,isvisible
shap,可选 arrow,turtle,circle,square,triange,classic
tilt，settiltangle，设置海归性质角度，不改变海归移动方向

使用事件

turtle.onclick(func ,btn=1,add=None)
turtle.onrelease(fun,btn=1,add=Nne),释放鼠标时执行函数
turtle.ondrag(fun,btn=1,add=None),拖动执行函数
