如何做决策-教练思维

发表于 2022-07-05 更新于 2022-07-06

发表于 2017-12-15
38-39 两周内容总结和个人一些感悟。

如何做决策

决定是一个选择倾向，感性的。决策是根据一定信息，做出一系列决定，达到既定目标。
一个完整的决策包含 5 个步骤：问题、目标、方案、结果、权衡。
问题：我们为什么必须做决定？
目标：什么是最好的结果？
方案：有哪些可能的方案？
结果：这些方案各自可能有什么结果?
权衡：权衡，做出一个收益最大的决定。
有限决策，用一句话概括就是—— 真实的生活中，人要做最满意的，而不是最完美的决定。不理解有限决策，容易造成分析瘫痪和完美主义，表现为不行动。
三问断绝犹豫:如果现在不行动，方案会失效吗？如果有更多时间，会有明显更好的方案吗？如果犹豫下去，会影响其他重要方案吗？
提问是决策的一部分，描述问题的方式很大程度上影响了决策。
提问就是目标拆分的过程。不断优化问题与目标之间的映射关系和路径并模拟演练。
我们容易迷失在问题里而忘掉最初的目标。人生就是一个从生到死的有限状态图，路径的权重和状态的连接数让人追逐。有人使用最短路径优先，有人选择遍历尽可能多的状态。
沉没成本是追不回的，只会消耗更多。假如你是另外一个人，刚才才重新穿越到你的身体了，重新面对自己生活，你会考虑停止做生命里的哪些事情呢？

教练思维

通过滑雪教练教打网球的例子表明：教练的主要工作是不断清晰学员的目标，减少学员实现目标过程的障碍。
粪坑里的 007 例子表明：一切后果都是自己有意的选择，自己要有承担选择后果的责任。吃自己的狗粮。
在清晰目标和清理障碍过程中，要懂得区分。区分感受和事实、区分事实和假设、区分事实和真相。
清晰目标过程中需要提问。提出度量行问题能具体问题，便于找到解决方案。

一台 centos7.2 的服务器，可以同时提供 yum 源服务和 apt 源服务。都是使用 apache 提供 web 服务。同步其他 yum 源服务器使用 reposync，同步其他 apt 源服务器使用 apt-mirror.

Apache 用于提供 web 服务。

招聘信息的关键字，逐一突破。

keepalived 是什么

就是一个可自己配置的软负载均衡，F5 等设备未包装前的样子。
keepalived 是一个 C 语言编写的路由软件。是 linux 系统上一个简单和健壮的负载均衡和高可用工具。负载均衡框架依赖于 Linux Virtual Sever 内核模块提供 layer4 负载均衡。keepalived 组件通过一系列健康检查来动态和自适应的维护并管理负载均衡的服务资源池。高可用通过 VRRP 协议实现。VRRP 是一个路由失效备源基础模块。keepalived 使用一系列 VRRP 有限状态机钩子提供底层和高速的协议交互。Keepalived 框架可以独立或一起使用来提供一个弹性的基础设施。

十万个为什么

发表于 2022-07-04

发表于 2017-12-12
提问是学习知识、融洽沟通、有效管理的好手段。对自我提问更能认识自我。

为什么要提问，好处在哪里？

提问能使注意力聚焦，专注做一些事情能增加思考深度，获得产出大。
提问能提高知识学习的效率。学以致用，提问就是给一个可检测的学习目标。在不断检测行动与目标之前的差距过程中获得反馈，指导下一步有效行动。
提问能提升沟通效率。在沟通中通过提问获取对方的沟通意图，根据对方目的沟通更能够达成一致。
提问能有效管理。管理是沟通的一个子集，通过提问发掘管理对象的中长期目标，在管理者认同目标后更能自我驱动，做有利于公司和个人的事情。
提问能提高创造力。创造力是一系列知识的组合，灵感迸发是跨领域知识组合，深思冒出是垂直领域知识组合，即兴发挥是知识和场景组合。提问是开启大脑搜索引擎的关键字。

如何提出一个好问题？

1 姿态要以学习者姿态，小学生。要敢问肯学，不做炫耀者。
2 提问要有套路，连续性问题深度思考。套路举例：最好的结果是什么？针对一种情况，下一步该怎么做？目标达到了多少？
3 提问前先准备。先问搜索引擎，排查初级问题。根据对方专注领域提问。提出问题要清晰，在有限的人生范围内尝试。

对自我提问能促进自我成长

我们内部有弹幕系统影响我们的行为。
自我积极对话，构建正向反馈的弹幕。以自己好友的姿态陪伴、鼓励、激励。
你经历了什么？
你从中学到了什么？
哪些地方做得不错？
有什么方法让失败变为转机？

心里边界

发表于 2022-07-04

发表于 2017-12-11
非常走心的一期，理解后可看明白生活中的很多行为背后的逻辑。

什么是心里边界

心里边界是自我意识和非自我之间的界限，存在于想象中。我们对自我现实环境的边界认识清楚，对心里边界往往忽视。心里边界在童年早期（3 岁以后）开始创建并慢慢形成。0-3 岁是一切皆我的状态，所有感受到的都是我的。3 岁以后开始区分“我的”和“别人的”，在尝试物品占有、情绪表达、说不要的过程中逐渐试探出心里边界。在开始求助阶段形成在有边界的同时与边界之外的个体融洽合作。

心里边界模糊会怎么

心里边界模糊会把感情投射到你无法控制的事情之上，就产生受苦。
心里边界不清的人，也不容尊重别人的心里边界，往往会让他人很痛苦。

如何识别心里边界越位

心里边界越位 4 中模式
第一种，不尊重别人的边界，经常控制别人，主动出击型越位。越位进入别人的边界，短期也许会让你获得掌控干，长期却是一个双输的事。
第二种，不尊重自己的边界，跨越自己的边界，导致完美主义。他们构想了一切外界的“应该”，却自己就是做不出来，是一种无所不能的无能状态。
第三种，因为边界不清楚，导致局面失控。不知道社交媒体的属性，社交媒体是广场，不是私有的。
第四种，心里边界过于僵硬。
最佳状态把自己想象成一座房子，既有独立的空间，也有待客的客厅，既不会完全关门不让人进来，也不会一上来就把人往卧室带；既耐得住寂寞，也有谈得来的朋友，既能帮助别人，也能接受人的帮助；既亲近，也独立。

重建心里边界三招

看破不说破。看破常见的一哭二闹三上吊的心里边界越位事情，不说破避免伤害。
区分伤害。明确界线，短期会有“伤”，会有不和谐的感觉，但长期是清清爽爽的过程。从别扭的拒绝到正当的拒绝，是一种修炼过程。你的界线清晰了，周围的人的边界也会清晰，时间利人利己的事情。
重建边界，从温柔的拒绝开始。为别人活是异常永无宁日，毫无胜算的自我恐怖主义。
