云原生时代的运维新归宿

发表于 2021-03-01 更新于 2022-06-29
  1. 背景：云原生技术快速发展和普及，已经进入云原生时代。
  2. 传统运维困局及原因。责任多，传承难。
  3. 云原生时代运维特征。基础设施即代码；运维特性前移到开发阶段聚焦基础设施层；基础设施可靠性和运维自动化。
  4. 如何在云原生时代运维？用软件工程思想去完成运维自动化。要进行云原生应用交付，需要云原生自动化运维平台。
  5. 运维转型：熟练掌握容器和 k8s，掌握 go 语言，关注 devops，云原生社区，学习使用开源工具解决问题。
  6. 云原生运维价值。助力业务更敏捷可靠。
  7. 生产力担当？好基础设施有利于创建好产品，快速迭代增加企业竞争力，技术开放降低生产成本，中台模式提升效率，快速试错、反馈增活力。

发给妹子的文章摘要汇总-8

发表于 2021-02-15 更新于 2022-06-29

基本上，如果想成为好作家，就必须做三件事：多多阅读，仔细深刻的倾听，以及多多地写。同时，不要想太多，只要长驱直入文字、声音和各种知觉的核心，并且让你地笔在纸上写个不停。
—《写出我心》

我这么试过，看了文学理论后，发现这是意识流。

教父里的人生观：
第一步要努力实现自我价值
第二步要全力照顾好家人
第三步要尽可能帮助善良地人
第四步为族群发声
第五步为国家争荣誉。
事实上作为男人，前两步成功，人生已算得上圆满，做到第三步堪称伟大，而随意颠倒次序的那些人，一般不值得信任。这也是古人的“修身齐家治国平天下”。所以，在你想要准备开始“平天下”的时候，也得先想想，自己的生活有没有过好，家人照顾好了么。。。

在聊职场八卦，不同工资女生睡觉前都在想什么时，提示要多想自己和周围的人，少操拯救地球的心。

生活方式以及是否追求财务独立的决定权都在于我们自己。我们有两种选择：一是随大流（成为市场营销者和销售人员的目标）；二是默默地追求我们的经济自由。
我们中 80%的人都认为伴侣在通往财富成功的道路上扮演非常重要的角色
经济独立属于那些愿意分配时间、金钱、精力和认知资源以实现财务目标的人。
《财富自由》

妹子，你愿意和我选择第二条路前行吗？

因为劳动收入转瞬即逝。那些获得财富自由的人靠的是储蓄存款和投入资本产生的非劳动收入。
收入是你今天带回家的钱，而财富确是你明天，后天以及今后每一天所拥有的钱。

我们也要增加非劳动收入。或者让劳动成果持续产生收入。

  1. 工作是一场马拉松，而不是短跑。
  2. 尊重人，而不是头衔。
  3. 工作从白板开始，而不是从键盘。
  4. 产出的是价值，而不是代码。
  5. 热爱生活，而不是工作。
  6. 学会享受过程。

网上看到的对工作的思考，你呢，最近有看到有意思的吗？和我分享一下啥。

发给妹子的文章摘要汇总-7

发表于 2021-02-15 更新于 2022-06-29

决定我们这一生的，并不是我们过去的经历，而是我么自己，赋予那些经历的意义。

人生意义的问答，你找到了几个答案？

让我把话说完
“快看，那个人还在动”。一个五六岁模样小男孩拉着他母亲的衣角，指着墓穴里的我说。
小朋友，你没看错，我偶尔还能动一下，还能思考。趁还有几口空气可以呼吸，我把用尽一生时光，对这个世界的认识，写一封信告诉你。有些内容你可能现在无法理解，不要急，慢慢你会明白的。

  • 人挣钱难，钱挣钱容易。因为人要睡觉，钱不睡觉。建议你去探索钱生钱的方法，让钱帮你挣钱，而不是你花费大量的时间挣钱。节省下来的时间，用来探索任何你感兴趣的事情。
  • 小孩子才讲对错，大人们只谈利益。世界是有灰度的，不是非黑即白。动画片、电视剧里我们能明显区分出好人、坏人，那是认为设定，为了惩善扬恶。好人也有坏的时候，坏人也有好的一面。不要天真，要仔细观察判断。
  • 要对这个世界不断释放善意，这个世界就会善待你。这句话是我从一个理财大 V 抄来的。善恶交织的世界，在善恶中间有广阔的灰色地带，不怎么善也不怎么恶，就是我看到的现实世界。相同爱好的人聚在一起玩。所以我建议你偏向善的一边，周围就会聚集善的人。在这种氛围中，生活得比较舒畅。
  • 天真容易被欺骗，但善良不是天真。

— 原创

有感于老俞的文章，躺在墓穴里听到最好听的一句话“快看，那个人还在动”

因为现实中所有事情都有风险，只要做事就得准备承受风险，一味躲开是不行得。什么风险都不冒得话，你就不该起床。不过还是得提醒你一句：床，是死亡率最高的地方。

富有不是高消费的生活方式，而是拥有规模足够大，能够产生收益的净资产。
— 《邻家的百万富翁》

写作需要明确目标，确定读者，用头脑风暴总结出中心思想，组织内容，写初稿以及修改。
—《完全写作指南》

写不出的时候，想想这六步。

我从许多失业的人哪里了解到，失去一份工作有着两种意义：一场可怕的灾难，或者一次机会，一次找到更加有意义，更加让人满意工作的机会。
然而我们不能忍受这种状况。于是我们就转移注意力，通过每日的娱乐进行消遣，比如看电视（可能还有其他毫无意义的活动，甚至吸毒）。我们看到电视中的演员，他们表演得就像他们也这么生活一样。现实生活就这样被肆意模仿了。当然了，空虚就以这样的方式暂时被填满。而电视的作用在于逃避那个明显不值得去面对的现实。
—《财务自由之路二》

奶头乐的复杂解释，不是很习惯。文章要写的清晰。

我们必须时刻准备着，为了未来的自己而放弃现在的自己。– 查理斯·杜博斯
现如今，最大的两大阶级是信息掌控者和无知者。未来的文盲不再是不能阅读的人，而是那些不会学习的人。
在信息时代，你不必用时间来换钱；你可以用想法来换取金钱并且一直赚下去，而不必投入新的时间。
旧规则：人们工作一次赚一次钱。有的人找一份尽可能高收入的工作，并努力工作，他们就会收入可观。
新规则：你可以利用一份工作获得多次盈利。请你富有创造力，敢于冒险，并且为了尽可能久的从一份工作中获得收益而勇敢寻路。

懂得的道理，有没有去认真寻找路径？
