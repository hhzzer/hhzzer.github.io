徐霞客和唐伯虎的故事

发表于 2016-03-07 更新于 2017-06-16

今天早会讲了这个故事，晚上回想起来，也是挺好的事情。

徐霞客和唐伯虎的故事还是要从徐霞客祖辈说起，徐霞客的祖父徐经与唐伯虎一年进京赶考，徐经有钱，唐伯虎有才，就结伴到京城。徐经想，大老远都跑来了也要捞点啥，就开始到考官处活动，最终获得了当年的考题。再想：自己知道了考题，获得满分难免引人怀疑，还要找人障眼。找到了唐伯虎不经意就透露了，唐伯虎也明白，知道后两人对笑不语。后来泄题被查，终身取消考试做官资格，唐伯虎就成了风流才子，徐经他曾孙也成了侠客。

徐霞客本名徐成祖，万历十五年出生。可能是受他祖父影响，不爱科举爱游玩。到处游玩也招父母反对，结果父亲去世早，母亲反对也弱，就放开了游山玩水。从20岁开始一致游历祖国大好河山到病逝，每年就过年回一趟家，真实敬业。徐家在他祖父辈还有钱买通出题考官，到他父亲这就没多少钱了。按照当时的物价水平只够他每年旅行一次，要执行他这么满的行程，只有穷游了。那个时候也没有什么冲锋衣、登山靴的，就粗布衣。没有饭店，在外行走都靠干粮，脖子上挂一布袋饼，说不定能吃上半个月。住就住野外、寺庙什么的，住不起官方招待所-驿站。行，就靠拼铁脚板了，放到抗日战争时期，应该是个特种步兵。徐成祖就这么较真的穷游二十载，一边游一边写，后来人读到他的路书，感动的把他名字都改了。

所有这些内容，都是在《明朝那些事》读出来的，这部长篇描述明朝280多年历史，描述了多少政治风云人物，如于谦、王守仁、张居正，却以徐霞客作为结尾，让我颇为意外。一读再度，也算了解了自己生活意义的一些解释：成功只有一个–按照自己的方式，去度过人生。

我为什么写博客

发表于 2016-03-07 更新于 2017-06-16

每当我发现购买的域名，搭建的博客很久不更新时，我就开始思考这个问题，每次都有不同的答案。
最开始看四火的博客上一篇《我为什么写博客》后心血来潮，就觉得我也要像他一样，写博客。用来回顾自己技术成长，记录唠叨。下了决定后就开始想在哪里写博客，csdn、博客园看的文章都比较多，但是好多独立博客的内容是最吸引我的，写独立博客的种子就中下了。先尝试了免费的，比如在新浪云上，使用一个wordpress模板，创建一个个人博客，虽然是二级域名，但是还是自己独立可控。在新浪云上写了几篇后，感觉不够方便，排版也不好看，就又开始寻找。看到hustlzp博客时，发现farbox是个好东西，写博客有客户端编写，自动同步到网络空间，且支持markdown格式，试用了1个月免费版后，思考两月，入了。写作工具有了，没有独立域名始终感觉不爽，遂开始寻找域名。在2016年初，思考年度计划时，痛定思痛，在万网上入得此域名，5年。有了折腾经历，有了金钱投入，我以为我就能坚持写博客了，其实不然。

今天一月写了上一遍博客后，又有两个月没有更新了，于是又开始重新思考：我为什么写博客？
思考的方向也类似，寻找我敬仰的独立博客大牛，今天找到了阮一峰。经济学博士的程序员，博客写的一样牛逼。回答为什么要写博客的答案可以用回答为什么要打电话一样。因为人渴望沟通。写博客的时候在与自己沟通，博客发布后读者浏览了实在与读者沟通。

自己写博客更新慢的一大原因是追求完美，有了一个想写些东西的冲动，但写出来的东西不通畅、逻辑不严谨，就不好意思发布出来。这一点要避免，因为成长是一个渐进过程，记录思考的过程也一样可以和自己沟通。

把时间当朋友 读后感

发表于 2016-01-25 更新于 2017-06-16

读书的目的是理解一本书，如果读完一本书之后，说我读过这本书和说我阅读了这本书，是完全不一样的两种状态。
李笑来「做时间的朋友」书中写道：理解一本书最重要的是使用它，若没有环境使用，就教导别人。知识，你会用了就是你的了；你没有条件用，但是你能教别人怎么用，你也得到了。最近罗胖和36kr都在引用这本书的内容，看来我要找时间买来阅读应用了。

hhzzer诞生第一天

发表于 2016-01-21 更新于 2017-06-16

今天过得还行吗？

今天是我网络ID诞生的一天，倒计时5年，构建网络中的我。

完成了哪几件重要的事情？

工作上继续处理服务器切换的bug，是由于没做好规划引起的，自食其果，应该吸取教训。以后做服务器切换一定要定义服务器状态，确保切换前和切换后状态一致，包括数据、配置等。其他就是接了年度员工大会现场IT设备保证的任务。

自己事物上完成了如下几件事情

  • 完成域名hhzzer.com的注册，以后再网络上就用此ID行走江湖。
  • 完成farbox的续费，时效5年。在这5年内要在这儿构建一个虚拟的我，真实真性情。
  • 看完罗胖的新一期视频《这个世界会好吗？》,主要分析《物演通论》里「递弱代偿」理论。记住的有一点：社会分工越来越细不可阻挡，兼职成不了气候，你只有降低适应行，提升专业化，才能获取财富。

    关心朋友和家人了么？

    和妹子微信语音聊了27分钟，都是各种唠叨。不过我觉得聊未来的话题太多了，让妹子不好接话有些不好。以后多聊现在和过往，未来的话题当面聊。
    今天收到姑姑寄过来的10斤纸皮核桃，很是高兴和感动。和姑姑通了电话聊了在外地过年的事情，有一丝忧伤。

身体怎么样，锻炼了么？

QQ运动7000多步，算是一般的运动量。只是今天天气特别冷，一回家就钻被窝，根本不想出来。古人云「夏练三伏，冬练三九」，这简直是反人类。

有什么新的感悟和收获？

看了best33和hustlzp的博客，又激起了我写博客的决心。我决定在网络上，记录一个虚拟的我。一方面来督促我成长，一方面总结自己。因为hustlzp博客中有一段话

我们其实每天都在睡觉，包括我自己，哪怕每天有很多事情也是在睡觉，没有真正在活着，朦朦胧胧就过去了。

我要在网络上刷存在感，不然每天就这么浑浑噩噩的过去，不留任何痕迹。我要在本博客、github上留下我的足迹。域名和博客需要花钱买，而且还有时间限制是好事，正因为如此才更能提醒我

逝者如斯夫，不舍昼夜。

看到月光博客上讲排版基础，以后写博客也要多多参考。markdown语法也不是很熟悉，以后要多所使用，优雅行文。
