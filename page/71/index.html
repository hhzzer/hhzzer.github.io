cisco snmp oid研究

发表于 2016-05-13 更新于 2017-06-16

最近使用zabbix监控cisco交换机相关情况，计划使用snmp协议获取监控指标数据，一边研究一边记录整理。
##配置snmp协议
首先需要在交换机配置snmp服务,其中xxx为自定义连接字符串，RO为只读。
hostname(config)#snmp-server community xxx RO
配置完成snmp协议后，就可以通过命令或工具，附带OID参数，获取感兴趣的数据。
##查询snmp信息方法
linux中可以通过snmpwalk 查询snmp信息。也可以通过Getif工具在window下查询，运行时注意使用兼容模式运行。

了解了配置snmp服务和查询方法后，要获取监控指标数据，关键是找到OID（Object Identifiters）,也称对象标识符。OID相当于通过snmp协议释放出大量数据组成数据库中的查询条件，输入不同OID，获得不同结果。上面例子中OID=’SNMPv2-SMI::enterprises.9.3.6.3.0’,就查询出设备序列号。

##获取OID方法

  1. 通过cisco官方网站查询
  2. 通过下载MIB（Management Infomation Base)库，直接在本地查询。
    一般是先通过网站，查询OID结构，然后再通过Getif工具查询设备，观察返回数据。

###常用OID整理

MIB和OID
OID(对象标识符），是SNMP代理提供的具有唯一标识的键值。MIB（管理信息基）提供数字化OID到可读文本的映射。
iReasoning MIB Brower 使用方法
下载全平台包link,解压后cmd到 mibbrowser\ireasoning\mibbrowser\lib 目录，运行 java -jar brower.jar 运行软件。
下载 cisco MIB包link,使用MIBbroswer加载包后执行。

《富爸爸、穷爸爸》读后感

通过在杂货店做3周枯燥的收入低的工作，学习不要为钱工作，要让钱为你工作。
现金流表和资产负载表认识。资产是能将钱不断装入你口袋的东西，负债是不断从里口袋掏出钱的东西。获得财富就是不断增加资产的过程。人生钱难，钱生钱容易。
打工挣钱和开公司挣钱由于税收不同，财富累积速度差异大。工资收入 纳税 消费，公司收入 消费 纳税。
工人付出最大努力以免于被解雇，而雇主提供最低工资以防止工人辞职。

如何做：处理好恐惧与贪婪的关系，给予后获取。

鸡汤虽好，不可贪杯。操作手法美国可行性高，国内不高。有些思路是值得学习的，再读一遍后整理到后面。

欢乐颂

发表于 2016-05-07 更新于 2017-06-16

很火的一部连续剧，今天终于看完了。整部剧节奏感强，剧情紧扣现实热点，能引起很多共鸣。
整部剧讲的是5个上海都市女青年成长的故事，一层楼的5个邻居有不同的出生背景，也造就了不同的性格，对同一事物的不同看法的冲突，持续推动剧情的发展。主要说说我从5个主角中获得的吧。

安迪 是个孤儿却很优秀，逻辑思维能力和分析能力都很厉害，虽说剧中被设定为感情方面欠缺，但人际交往能力还是一流的。早上晨跑，每天阅读2小时，多年自律养成的良好习惯才有了今日的辉煌。清教徒的信仰：努力过好自己的生活，若有余力，帮助他人。

樊胜美 职场人精。10年的工作把她磨练得随大溜，获取安全感。一大家子的烂事看得直叫人喊虐。装坚强欺骗别人也欺骗自己，但若不这样强撑，恐怕早就逃离了。后几集剧情的真情与前面的装形成巨大反差，做回真正自己的樊大姐还是很招人喜欢的。

曲筱萧 富二代美女，行走江湖练敏锐洞察力和应变能力。看人看事准，分析起来也头头是道。关爱流浪猫增加爱心值得剧情在我看来有些端庄。与各色人交往多了，练出这样个机灵鬼。

邱莹莹 一根直肠通大脑。上当受骗哭，找到工作笑，这样一个对生活做出直接反应的人，算是活出真正的我吗？最后获得了小小的幸福，就是大多数普通人的期望吧。

关雎儿 邻家多才妹妹，努力工作，向优秀的安迪学习，实习期典型性格吧。沉默越久爆发越厉害，她是2202中最有可能爆发的。

5个主角中够有我喜欢的点，最开始喜欢安迪，小曲的聪明，后来喜欢小蚯蚓的倔强而充满希望的生活方式。安迪、小曲达到了看自己的格局，其他三，随大溜。

一只中华田园犬在五一节的自白

发表于 2016-05-02 更新于 2017-06-16

舔了舔尾巴上的毛，感觉五一假期就要来了，真是好期待。院子里会回来好多人，上学的、在城里工作的都回来了。特别是上学的学生回来之后，各家都会炖点鸡鸭排骨什么的，我也就机会改善伙食了，最近的饭一点油水都没有，叫都懒得叫了。
这不，看见院子里新停了一辆车，我把耳朵贴地上一听，就知道是邻居家的少爷回来了。我得打起精神，给他摇几个漂亮的尾巴，第一印象很重要嘛。果然，在我看见他上台阶后，蹦蹦跳跳的摇着尾巴欢迎他，他也过来摸了摸我的头，我就知道这招对付他管用。可能他看我被绳子栓得可怜，就把绳子解开了，把我高兴得直跳。
获得自由后的我很兴奋，到处探索。以前我想去的几个地方都去闻了个遍，确认一下是否还有骨头。
