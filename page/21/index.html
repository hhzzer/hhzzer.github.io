配置工作用centos7系统

发表于 2022-07-04

发表于 2017-08-02
工作网络和自带设备网络隔离，需要使用堡垒机。堡垒机为 windows 系统，很多工具在 linux 下使用方面，故需要早 windows 机器上创建安装一个 linux 工作环境。

操作环境配置

  1. 调整虚拟机版本为 9,并修改配置
    堡垒机为 vmware 虚拟化的虚拟机，版本为 8.要在虚拟机中嵌套虚拟机，需要把虚拟机版本升级为 9 及以上，并在 web 管理界面开启虚拟化支持。参考链接
  2. 安装 vmware workstation
    下载链接
  3. 配置虚拟机资源 1C1G10GB,网络选用 VMnet8,配置端口映射：内部 22-外部 2222。
  4. 安装 centos7 系统。时区 Aisa/Shanghai basic .安装后无法启动图形界面
  5. 设置 yum 代理 vi /etc/yum.cfg “proxy=http://192.168.x.x:port"
  6. 安装软件 yum tree ansible r

ubuntu上设置chrome代理

发表于 2022-07-04

发表于 2017-07-30
使用某灯后，chrome 浏览器无法联网，提示检查代理。使用另一个浏览器能正常联网，就 bing 答案，最终解决
搜索“network”，在 network proxy 中选择 none。
另外可以设置代理的地方哪个有
.bashrc, .profile, /etc/enviroment，/usr/share/applications/google-chrome.desktop

《白玫瑰.红玫瑰》读书笔记

发表于 2022-07-04

发表于 2017-07-27
吴军书单的缘故，速读此书。
张爱玲的小说读起来是轻松的，读后的思考是持久的。
白玫瑰讲一个外国留学生和女人有关的故事。巴黎的 firstblood，英国的坐怀不乱，回国后与同学妻子暧昧，结平凡妻，婚后家庭生活不顺后的放荡，最后悔悟?
红玫瑰讲一个年轻离异的小姐，寄居在哥哥家受嫂子闷气。给 7 妹介绍婆家时遇见放浪不羁的留学公子。舞会初识、香港热恋、回上海试忠心、再往香港经历战争见真情，在一起。
老婆孩子热炕头，抱着白玫瑰平凡过一生，是一种爱情。基因里的好色与不安分，不停鼓噪男人外遇，寻找不一样的红玫瑰。
男人找伴侣要找聪明，有魅力的。女人找伴侣要找对自己好的，而且是行动上。
武志红在直播中讲边界问题，物理上的、身体上的、心里上的边界。处理人际关系，即为划界限，在不同的界限范围，做不同的反应。
罗辑思维产品试金石：1）是否给用户带来价值 2）用户能否听懂 3）用户读过后能记住什么 4）用户获得后是否有意愿，有能力对朋友讲述。
我读了这本书后，对我的价值是认识到男女都躁动不安。我读懂了么？这需要和别人讨论或看书评，拿自己的结论和别人对比。我记住的是什么？哪个少年不多情，哪个少女不怀春。我怎么给别人讲？第二个故事是第一个故事的续集。
