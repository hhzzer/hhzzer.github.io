《网络迷踪》观后感

发表于 2018-10-26 更新于 2022-06-29

看见左耳朵耗子在微博上推荐此电影，看后总结下。

这种悬疑类型的电影最近看了不少，暴裂无声、心迷宫、看不见的客人等。

梗概

父女之间的沟通都通过电子信息，比如电话、短信、Facebook，没有面对面沟通。90%的镜头都在电子屏幕上。
一天晚上，女儿给父亲打了 3 个电话，父亲没接到，回拨过去无法联系，报警女儿失踪。女警官主动接手案情调查，希望父亲配合提供所有相关资料。父亲发现对女儿一无所知，不知道她最亲密的朋友、爱好等。还好有女儿小时后的网络日记和女儿的电脑。获取女儿的社交网络登陆信息后，开始从女儿账号中了解女儿。知道她在母亲去世后不喜欢钢琴，去直播，和同学参加课后学习小组。
女警官跟踪信息，调查因果。父亲自己也调查，之后发现女儿车沉没的湖。女警官侦破案件，表明被害女孩为一个精神病所杀，然后精神病自杀，一切都证据源于精神病自杀前的自首视频。
女儿网上追悼会，父亲发现蹊跷，觉得真相不止于此。调查女警官关系，发现女警官一直在回避，不想找到真相。后女警官在证据面前坦白，为了保护儿子，故意引导案情调查，一切都是她的计谋。
在女儿真正失踪地点找到女儿，医院抢救，生还。

感想

一直通过网络沟通，导致父女关系冷漠，变成熟悉的陌生人，不推荐。面对面沟通无法被替代。
一步错，步步错。溺爱等于伤害。
怀疑，独立思考是重要的能力。

Pro Git阅读摘要

发表于 2018-10-22 更新于 2022-06-29

git 命令零零散散的用，是时候需要一本书系统了解了。

摘要

  • git 和其他版本控制系统最大的区别是对待数据的方式上。其他版本控制系统是基于单个文件做版本控制，git 是将数据看作小型文件系统的一组快照。

  • git 有三种状态，你的文件可能处于其中之一：已提交（committed）、已修改（modified）和已暂存（staged）。已提交表示数据已经安全的保存再本地数据库中。已修改表示修改了文件，但还没有保存到数据库中。已暂存表示对一个已修改文件的当前版本做了标记，使之包含在下次提交的快照中。

  • git config 用户配置 git 外观和行为。变量存储在 3 个不同位置 1）/etc/gitconfig 针对系统上所有用户配置，采用参数–system。2)/.gitconfig 或/.config/git/config 针对当前用户配置，–global。3）仓库目录.git/config,针对仓库配置，git config 不加参数。变量高级覆盖低级，3 覆盖 1.

  • git 文件状态切换关系

  • git add，是个多功能命令，添加内容到下一次提交。可用于开始跟踪新文件、把已跟踪的文件放到暂存区，合并时把又冲突的文件标记为已解决状态。

  • git rm，–cached，将文件从 git 数据库中移除，在工作目录中保留文件。

  • 分支查看  git log –oneline –decorate –graph –all

  • 分支创建、切换、合并

  • git 工作流   主干开发，分支发布。主干发布，分支开发。

总结

看完书和阮一峰的几篇文章，解惑一下几点

  1. .git 是 git 项目的数据库，存放已提交信息。基于这个数据库，可以恢复或生产不同节点的文件。

  2. 分支是 git 和其他版本管理对比最大优势，源于提交对象的指针移动。

  3. 分支合并需要用到时再多练习

暴裂无声-观后感

发表于 2018-10-15 更新于 2022-06-29

周末走亲戚，教电子琴书用法，跑步，看暴裂无声 2 遍，最想回顾的还是这个电影。

刚开始看有点恐怖，没有人物对话，全是背景音乐，有一点压抑。剧情跳转很快但简单：矿区一个煤矿工人的儿子-磊子，放羊人失踪 🐏 还在，矿工张保民回家找儿子。找儿子过程中碰见矿霸占矿，打抱不平遇见大矿老板昌万年。许文杰律师帮昌万年打官司，大事化小非法开采问题，获得 50w 现金。随着情节发展昌万年为了要某个证据绑架律师女儿，张宝民误抢到以为是自己儿子。

G 点：律师在交代罪行末尾，只交代非法采矿事情，然后带起眼镜，说：没有了。谷风村一座小山爆炸，坍塌。

看后能让人思考的电影，才是我喜欢的。视听刺激，对我触动不大，因为我是慢热。

张保民，保不住儿子，自愿哑巴。

《赋能》读书笔记

发表于 2018-10-12 更新于 2022-06-29

全书分为 5 部分，1）当前社会不确定性大增，旧的应对方式不再生效，组织需要调整。2）化繁为简，发现成功团队中的秘密。3）信息共享，空间深井的打破，一个组织内部如果具有透明性，并且拥有有效的沟通机制，就能产生意想不到的效果。4）组织中赋能的历史，优点和必要性。5）赋能的效果。
先写概要，发酵一会再展开。
原创的狗屎也是狗屎。

听了樊登的讲解，对此书有了更近一步的了解。
全书就将一件事：面对不确定的时代，需要打破深井，共享组织目标，权力下放，才能实现组织目标。
为什么？
对于伊拉克恐怖组织，使用深井的组织架构被打败了，所以深井病得治。
逻辑演绎：你们组织在商业上面临得情况和我在伊拉克战场上面临得情况一样，所以你们也得治病。
我们怎么治病的？共享目标和过程信息，员工根据这些信息制定自己目标，完成目标，推动组织目标的实现。部门之间特派，使用特派员打破部门之间的壁垒。
