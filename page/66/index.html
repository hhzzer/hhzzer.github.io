发表于 2016-12-23 更新于 2017-06-16

“但有一件事我必须说说，作为一个表演着，我为五万人表演过，也为五十个人表演过，我可以告诉你的是，给五十个人表演更难。因为五万人会有单一的人格，而五十个人不会。每一个人都会是一个个体，有独立的身份和对世界不同的看法，他们会更清晰地感知事物，会检阅你的真诚，以及你如何用自己的才华将其表达。” – 鲍勃.迪伦

《超级个体》观后感

发表于 2016-12-17 更新于 2017-06-16

邪恶冷笑着，就这样;而善良不停的在问，为什么？ –《战后的平静》 托马斯.哈代
昨天看了罗辑思维新的一期《古典：超级个体》，今天写一下讲的主要内容和个人感受

要讨论的问题

什么样的人能在这个时代活得成功，活出自我，为什么要成为一个超级个体？

开篇先用现实生活中典型的四种人，表明这四种人与目前这个时代存在冲突

  1. 模仿着。经常在媒体上看见各种成功着，商业上的成功，赚取大量金钱。我也想成功，就去学习他的专业技能，出事方式。媒体上也经常出现诸如《成功认识的7个习惯》等励志书籍，似乎学习了他们自己也能成功。但真的是这样吗？成功是一个复杂的过程，影响因素很多。就算成功人士不断公开回答他们成功的秘诀是勤劳、勇敢等，其实他们成功的或许并不是他们认为的因素，因为存在一个“幸存者悖论”。根据飞回战机弹孔分布，就得出结论要加固弹孔密集的区域，以期待飞机更安全。我们忘了考察那些被击落的飞机，或许是由于被击中了油箱，其他地方完好无损，也不能飞回来了。关注那些失败的原因，或许更有效。时间也是成功的一个重要参数。滴滴打车就在移动互联网和智能手机普及的情况下飞速发展，而同行先行着已经尸骨无存。所谓：“天时地利人和”。结论：成功很难复制，模仿着难成功。
  2. 知识瘫痪者说的是知识长颈鹿。热衷于刷技术前沿、行业趋势及国际形式等各种知识，眼界很开阔，但脚步原地走。知识转化为能力除了知晓，更重要的是练习实践。这种类型病因在于：我们很容易被外界的各种新刺激吸引，内在刺激就很难驱动行为了。
  3. 白日梦想家是指整天幻想，不去行动的人。行为表现为“计划”-“路演”-“转移”。做好计划，然后兴致勃勃给别人说，到头来不做，借口”计划有变“。开始做梦就会在意识中形成已经完成的信号，然后压力降低，动力不足，没完成被别人质问就转移目标逃避现实。
  4. 过份追求安全感的人，又期待安稳又羡慕别人的精彩。船在港口最安全，但那不是造船的目的；人在家里最安全，那也不是生活的意义。
    以上四种生活心态的人，不是和自己冲突就是和现实冲突，过着忧心忡忡的生活。那如何才能摆脱困境，活得安然呢？先要分析产生困境的原因。

我们这个时代已经步入人均年收入$8000门槛，因此幸福增长不能只靠钱，还要靠其他维度的满足才能获得更大的幸福感，类似与马斯洛的需求金字塔。其他维度就是自我价值的实现。怎么实现自我价值呢？

利用git解决hexo博客多PC间同步问题

发表于 2016-12-15 更新于 2017-06-16

好多次在家想写博客，都是由于不知道怎么在多台电脑上写hexo博客而终止。最近立下写文章和写代码的志向，多电脑写博客必须搭建了。

看了网上的一堆文章，选了一个基本靠谱的测试。基本思路

  • hexo 博客目录所有文件去除掉一些敏感配置（主站_config.yml和主题_config.yml)公共文件，其他可用的变化文件都同步到独立的一个仓库去。我的.gitignore如下
  • 生成博客hexo d -g后，git add . git commit -m "something",git push,同步到远程仓库。
  • 换电脑后，安装好hexo后，创建博客目录blog.
  • 删除原有的.git目录，初始化git仓库并添加远程仓库,并将主站配置文件和主题配置文件放置到指定位置。
  • 每次写博客三个步骤 pull blog，hexo new post ,hexo d -g ,git add ,git commit.

参考文章：http://chitanda.me/2015/06/18/hexo-sync-in-multiple-pc/
