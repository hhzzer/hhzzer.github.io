《别闹了、费曼先生》读书笔记

发表于 2022-07-04

发表于 2017-08-31
在万维钢和 marix 博客上都提及这本书，三天看完。
此书是费曼的自传，以给儿子讲故事的方式讲述费曼的经历。儿童时期擅长修收音机。上学时自学数学，发明自己的符号系统，数学运算能力强于他人。大学期间学习理论物理，在贝尔实验室实习。博士毕业后进入曼哈顿计划，解决计算问题。为军队工作期间擅长修理计算机、开锁（机械锁和密码锁），经常和别人开玩笑。曼哈顿计划结束在康奈尔大学教书，年假期间到巴西教书。批判巴西没有真正的科学研究，只有背诵。学习桑巴舞，达到业余高手参加比赛。学习葡萄牙语，用葡萄牙语讲课和汇报。学习绘画，能鉴赏作品，买画给按摩院，开个人画展。最后说了获得诺贝尔奖的过程和一些生活上的改变。
玩闹，就是费曼的一生写照。不停尝试新东西，不断学习，挑战自己的极限。
费曼学习法有必要再写一遍
要具有超强学习力，需要从输入、处理、输出上有进一步的认知和提高。处理信息有两种方式，1 是记住名称，2 是理解事情本质。
四步学习法
第一步，选择一个概念。这适用于任何学科的任何概念。
第二步，教授这个概念。把你的教授对象定位为小学生，用他们能听懂的语言去解释这个概念。
第三步，返工。这一步针对的是第二步你解释不清的地方，在返回到原材料中，重新学习。当你可以解释清楚后，合上课本，继续第二步教授这个概念。
最后一步，回顾和精简。把你刚刚对于概念的解释写下来，然后大声读出来。如果有不够简洁或者听起来让人困惑的地方，那就再一次精简，用最少的文字去解释清楚。而这时，你会发现自己真的理解了这个概念的实质。

又一次配置 zabbix server 的邮件报警，配置了半天，这次总算理顺了。

zabbix3.0 告警方式

zabbix3.0 之后支持告警方式有：电子邮件、脚本、短信、jabber。
电子邮件即使用 zabbix 系统自带的电子邮件配置模板，使用公共电子邮箱（126,qq）发送邮件。3.0 之前电子邮件方式不能添加有密码的邮箱，所以网上大多数教程都是使用脚本方式发邮件。3.0 版本使用电子邮件模板配置发邮件也不容易成功，建议不用此种方式。
脚本是使用服务器上的命令发送邮件。linux 系统常见发送邮件命令有 mail 和 sendmail。整个邮件传输过程有三个角色 MUA（mail user agnet）、MTA（mail transport agent）、MDA（mail delevery agent）。使用公网邮箱发邮件，只需要一个 MUA 即可。微信报警也是通过脚本方式实现。
短信需要接短信猫，jabber 不知道是什么。

操作系统：centos7.2
前提条件：服务器能连接外网 126 邮件服务器。

邮件发送过程分析

MUA 编写一封邮件–MUA 将邮件传递给 MTA–MTA 将邮件放在 MDA 保存-用户 2 使用 MUA 登录 MDA 查看邮件
postfix 可以担任 MUT、MUD 角色。可以使用 postfix 服务，给本机另一个用户发邮件，例 echo “mail content”|mail -s “mail subject” user1@localhost。要使用 user2 登录，而不能 su 切换到 user2,才能查看到邮件。

小富靠运，大富靠命

发表于 2022-07-04

发表于 2017-08-22
吴军老师这两天的来信，认真听了，能得到很多。

结论

小富靠运，大富靠命

证明

  • 为什么小富靠运气？

同门清华师姐高考运气好，本来上北京科技大学后来上了清华大学，运气好。后来的发展泯然众人，是命中注定。
美国通过中彩票获得大量才富的人的和在中国股市早期获得财富的探险家，后来都基本上败光，这也是运气好，命不好，无法大富。

  • 命的定义及获取财富的方法

每个人对命的定义不同，吴军老师的定义我比较赞同：命运是环境因素和个人看问题、做事方法绘制的两条线。获取财富的方法是信命、认命。
相信改变自己看问题、做事方式能改变命。如果由于长期行为累积出的习惯无法改变，就认命，不妄想。

  • 为什么大富靠运

Z 君看似经历一系列好运气，高盛、谷歌、亚马逊上市，但这都是由于他看问题、做事情方法给他带来的，命好是原因。出生只是运气，开始的好运气能让命好的概率大一点，但命运就像马拉松，塑造命运的过程才是跑完获胜的关键。
反证：李广难封，是由于命中注定。他又好运气可以上阵杀敌立军工，但都是已做士兵的心态行动。其实是不够资格封侯。

实践方法

练习对“扇耳光”的反应，有可能可以改变命运。三种反应：1）扇回去，2）认怂，3）分析该不该被扇，该就改，不该就给对方教训。
日常的谈吐可以看出一个人的命运，就是看他怎么反应，举例袁腾飞判断他学生妈下岗。
三同学 A，B，C 聚会，分出三类人。A 官员，被扇后思考，加入强势阶级，有权扇人。B 教授抱怨认怂，不认命也不反抗，必然结果。C 学者，思考后强大自己，让人无法扇到。

我的思考

以合作者姿态做事，与信命者共事，自私的自己把握命运。
