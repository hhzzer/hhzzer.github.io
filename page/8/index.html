要以玩为赢的姿态工作生活，而不是玩为享乐。

why

玩能增加经验值。玩是提前做生存预演，演练多了在现实生活中处理事情更有经验，让你更卓越。

CMDBuild搭建与使用

发表于 2022-07-04

发表于 2017-12-02
做运维一直想有一个靠谱简单的 cmdb，知乎上说 cmdbuild 好用，今天搭建测试一下。

结论

不好用，复杂，学习功能需要花费大量时间。
java 开发，不熟悉，不喜欢。
下一步还是使用蓝鲸配置平台或者 adminset

搭建过程

环境准备

操作系统 centos7.2，数据库 postgresql-server9.2,jdk 1.8,tomcat8

下载 cmdbuild 并部署到 tomcat

配置

设置 plsql postgres 密码后启动 tomcat，访问http://ip:8080/cmdbuild/ 即可通过 web 界面配置

遇到的坑

1 tomcat 启动不在 tcp 8080 端口监听，只有 tcp6 的 8080 端口
解决办法：禁用系统 IPv6，参考link
2 postgres 数据库连接失败
无法通过命令 plsq -U postgres -h 127.0.0.1 -p 5432 -d database 登录数据库
通过修改 postgres 的配置文件解决 vi pg_hba.conf ident 替换为 trust link
3 web 界面初始化时保存最后一步保存配置失败
通过查看 tomcat 的 log，发现是无法创建数据库，原因为之前创建过相同名字的数据库，修改一下数据库名称为不一样即可。

人际网络-个人品牌

发表于 2022-07-04

发表于 2017-11-29
30,31 周总结

人际关系

是什么

人际关系的本质是互惠，所有交往的活动都是基于互惠原则。

为什么

你的价值由你的朋友决定，人脉的目的是与高手过招，与伟人同行。
与他人一起学习成长更快。独学而无友，则固陋而寡闻。

如何做

要有持续的好关系，首先个人得有差异化的价值，才能惠及他人。
个人差异化价值可以考虑行业资深、资源链接、决策外脑、情绪友人四方面发展。
人脉要提前建立。之前认识后帮忙叫互惠，认识后马上帮忙叫互换，互换成本高于互惠。
与高手过招，与伟人同行。
做高成长、高价值、健康教育相关事情，使自己能力和牛人差异化，有机会能力互补。
聪明的让牛人帮小忙，之后回馈建立联系。
网络社会联网学习。跟随牛人，整合领域外人，联机伙伴。同一个问题通过与他人交流获得不同角度的认识。

个人品牌

为什么需要个人品牌

个人品牌能提高你的议价
个人品牌能降低你的交易成本
个人品牌能让你更自由：为职位、为钱、有能力做自己事情。

品牌的本质是什么

个人品牌包括品质和名牌。品质是指持续的输出高质量能满足他人需求的东西。名牌是有辨识度的影响了很多人。
品牌=价值 × 人数 × 频次。

如何创建和传播个人品牌

品牌创建关键要有远景识别度，很远的人能快速分辨出来。
创建个人品牌三要素：做对他人和自己有价值的事，品牌要简洁，要有同行认同。
品牌传播
高频传播，每一个行动都在传递的你个人品牌。
体验即品牌，做出好的产品，好产品就是你的品牌。
自我介绍框架化，不断强化别人的认知。
品牌最终传递的是个人价值观，为什么到这个世界行走 80 年？
