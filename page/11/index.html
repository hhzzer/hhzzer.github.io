自我管理和高效能

发表于 2022-07-04

发表于 2017-11-14
昨天胡乱总结，今天认真听了，还是老师总结得好。
所有行动或目标期待都源于人的能动性，所以一开始要认识自己。选择比努力更重要。

  1. 断舍离，清晰判断什么事是重要的。
    断，断掉不需要的东西；舍，舍去多余的东西；离，离开对物质的依恋。通过对东西的断舍离，找到情绪告诉我最重要的东西。通过对事情的断舍离，找到价值观告诉我的最重要的事情。真心认为重要，才有做事情的动力。断舍离是一个观察自己价值观并让价值观逐渐清晰的方式。
  2. 三件事和结果思维，把事做得有价值。
    每天记录要达成的三个结果并加上时间累计标记，发布到公开或半公开的地方，利用为自己保全面子的冲动去完成结果。有价值的事是有客户价值的事，客户愿意掏钱或掏时间和你交换。让结果更有价值的可从显性化、可交付、可复制三个角度努力。
  3. 心里比对和如果那么，提前制定做事的策略。
    做事之前的策略很重要，凡事预则立，不预则废。心里比对是把事情最好的结果和最差的结果对比，让我们避免盲目乐观而轻视要做的事。也让我们进一步筛选事情是不是头脑发热的冲动。
    如果那么就是在做事情之前想好可能发生的意外情况并为意外情况做好预案。等到意外发生时再做选择，大脑容易选择懒惰。多做如果那么规划容易把事做长久。
  4. 碎片化时间管理的加减乘除方法。
    碎片化时间不可避免，用合适的姿势拥抱是最佳策略。加，增加碎片化时间的价值。1 分钟做舒展运动，5 分钟做策划，每个碎片都应有产出。减，减少事件切换。将同类型事情累在一起做，打电话、回邮件批量执行。乘，多人有效的事情一起做，比如结队编程。学习效能金字塔指出调动感官越多，学习效率越高。独学而无友，则孤陋而寡闻。除，把大目标打碎分解，消除目标恐惧，阶段性获得正反馈有利于事情推进。
  5. 有时间也要有状态，精力需要管理。
    精力有限，有浓度曲线，在精力高时做高消耗精力的事，低精力做低精力消耗的事。精力可恢复，可锻炼。平时使用不要过度消耗，做了极限尝试后要花足够时间恢复。

信息就是金钱

发表于 2022-07-04

发表于 2017-11-10
一直吃信息闭塞的亏，难道真的要去信息风暴中心，北上广以及纽约么？
大学在偏僻的西北，校园里面除了学习，就是游戏，商业和技术氛围全无。技术上只知道书本上的，外面技术发展成什么样子都不知道。商业上只有忽悠学弟学妹们办回拨电话卡、买被子、日常生活用品，赚取低廉差价。没有商业就没有技术上的需求，技术研究和学习就没有方向。没有方向，游戏度日，浪费了太多的时光。
第一份工作在偏远的重庆城乡结合部，活动范围在厂区、宿舍，偶尔逛街要坐车 1 小时到最近商圈，信息接收少。工作之余小有收成就是定期买书并读完、晨跑长江大桥。结果还是决策失误在偏远地区买房，错过核心区域的房价上涨的红利。
痛定思痛跳出舒适区，真正进入互联网行业。每天通勤时间 3 小时，跟随年轻人挤公交、换地铁。上班地点为市中心，信息密度高，接收大量信息。公司后面公寓 8000 没买，现在可能涨到 18000.
以后要选择信息密度更高的地区呆，深入其中才能全方位感受，然后做出有利判断。充分利用互联网途径接收信息，补充身体感受的不足。要致富，走信息高速路。

谈读书-培根

发表于 2022-07-04

发表于 2017-11-09
通勤听了价值投资 2 篇和几篇散文，很喜欢培根的《谈读书》，全文搬迁，不知会不会有版权问题。

价值投资 4 观点

  • 买股票是拥有好公司的一部分所有权，而不是投机工具。养猪和养娃的区别。
  • 市场只提供交易机会，不反应出真正的价值。因为人是非理性的。
  • 留出安全边际，只买价格大大低于公司内在价值的股票。
  • 投资人需要建立能力圈，有所能有所不能。
    通过长期努力和学习，能够对某些公司，某些行业获得超出于几乎所有人更深入的理解，能够对公司未来长期的表现，做出更准确的判断。

谈读书

读书足以怡情，足以傅彩，足以长才。其怡情也，最见于独处幽居之时；其傅彩也，最见于高谈阔论之中；其长才也，最见于处世判事之际。练达之士虽能分别处理细事或一一判别枝节，然纵观统筹，全局策划，则舍好学深思者莫属。
读书费时过多易惰，文采藻饰太盛则矫，全凭条文断事乃学究故态。
读书补天然之不足，经验又补读书之不足，盖天生才干犹如自然花草，读书然后知如何修剪移接，而书中所示，如不以经验范之，则有大而无当。
狡黠者鄙读书，无知者羡读书，惟明智之士用读书。然书并不以用处告人，用书之智不在书中，而在书外，全凭观察得之。
读书时不可心存诘难作者，不可尽信书上所言，亦不可只为寻章摘句，而应推敲细思。
书有可浅尝者，有可吞食者，少数则须咀嚼消化。换言之，有只须大体涉猎者，少数则须全读，读时须全神贯注，孜孜不倦。
书亦可请人代读，取其所作摘要，但只限题材教次或价值不高者，否则书经提炼犹如水经蒸馏，味同嚼蜡矣。
读书使人充实，讨论使人机智，作文使人准确。因此不常作文者须记忆特强，不常讨论者须天资聪颖，不常读书者须欺世有术，始能无知显而显有知。
读史使人明智，读诗使人灵秀，数学使人周密，科学使人深刻，伦理学使人庄重，逻辑修辞之学使人善辩；凡有所学，皆成性格。
人之才智但有滞碍，无不可读适当之书使之顺畅，一如身体百病，皆可借相宜之运动除之。滚球利睾肾，射箭利胸肺，漫步利肠胃，骑术利头脑，诸如此类。
如智力不集中，可令读数学，盖演题须全神贯注，稍有分散即须重演；如不能辩异，可令读经院哲学，盖此辈皆吹毛求疵之人；如不善求同，不善以一物阐证另一物，可令读律师之案卷。如此头脑中凡有缺陷，皆有特药可医。

生死学的一周

为什么要谈死亡

价值观源于人生观，人生观最大的问题是对死亡的思考。我们必须有清晰的价值观，日常行动和决策时才有自己的依据。清晰的价值观源于对人生的看法，你这一生要如何度过，如何生活、如何死亡。吴军的向死而生，我们终究要死，那我们就要好好思考每一天如何活。

生死是什么

生死就是生的时候对死是什么感觉，如何面对。生的时候对死亡会有焦虑，产生焦虑的原因是 1）死得痛苦 2）死后归属 3）后悔当初。

怎么面对死亡

直视骄阳。
害怕死得痛苦有三种解决方案，生前预嘱、安宁疗护、获取疾病知情权。在选择于尊严网站可以通过选择题的方式立定预嘱。安宁疗护就是不用生命维持系统，自然的死去。获取疾病知情权有助于正确判断。
死后归属问题也有三个解决方案：繁衍后代、奉献到集体、创造自己活着的证据。我今天活着，所以我的博客还可以更新。我以后死了，后来人也可以看到我在 2017-11-08 的博客记录。个体已死，文字永生。
后悔当初怎么办？跟每天一期一会，向死而生。不是到某个年龄突然死去，而是每天过完都死去一天。这要死去的一天改如何度过，选择做什么非做不可的事情？
