全书讨论的是甜甜圈中间的孔，就是个人无形的品格。7 要素

  • 友善待人
  • 勇于承担
  • 发现他人优点
  • 助人为乐
  • 心怀感恩

Centos7.5安装iNode Client

发表于 2019-01-04 更新于 2022-06-29

问题

连接网络需要 iNode Client，记录 Centos7.5 x64 安装 iNode Client 过程

过程

1 下载 iNodeManager,link
2 安装 iNodeManager

3 运行 iNodeManager,生成 iNodeClient。

在图形界面中勾选 SSL VPN，生成支持 SSL VPN 协议的 iNodeClient。如果知道 ssl vpn 网关地址，可以在高级定制中输入 ssl vpn 地址。
4 将生成的安装文件拷贝到制定目录，安装

5 运行

报错处理

1 运行时找不到 libpng12.so.0，通过安装 libpng12-1.2.50-10.el7.x86_64 解决。第一次安装成了 libpng12-1.2.50-10.el7.i686,导致“wrong ELF class：ELFCLASS32”报错

2 启动 client，连接时报错“pango-Critical”
通过重启电脑解决

2018年终总结

发表于 2018-12-29 更新于 2022-06-29

2018 年最后一个工作日，总结过去，展望未来。

总结 2018

2017 年末总结时的 2018 年计划

财务目标：在 6 月份换工作，对比上半年和下半年，增幅 67.9%。基本达标。
健康目标：没有坚持，没有数据记录，可能断断续续晨跑了 20 天，每天 30 分钟。不达标
家庭目标：达标，只是 2019 年才正式宣告。
个人目标：电子琴购入，但没有练习。老是以缺琴架、乐谱、妹子不想玩为接口。不达标。
其他大事记：换房，通勤时间每天减少 2 小时，身心较为愉悦，但是没有充分利用这花钱买的 2 小时。
学习、阅读、写作：订阅了技术专栏 2 个，泛泛收听没有有效吸收。阅读了不少书，写了 13 篇读书笔记，剩下可能有一半的没写读书笔记。内网写了好多过程文档，博客文章全年写了 57 篇。
python django 入了门，使用 django，集成一个 markdown 编辑器，做了一个简单 wiki。
linux 系统操作熟练了，编译程序、根据日志、google 排查故障 ok，深入原理还没有。

展望 2019

2018 年时间还是溜得悄无声息，需要重读《奇特的一生》，在这段时间旅行上刻画更多的痕迹。
工作的一切，都是为了生活。要想过得有点趣，生活总得带点绿。健康的身体、有益的爱好是必要的。
财务目标：double，通过投资自己，投资资金等方式。
健康目标：跑步，每周一、三、五，除非下大雨影响跑步，都要去，不能因为天气太热、太冷等因素放弃。156 天，带上手环年底看运动总结。
趣味目标：

  • 电子琴增强练习

  • 英语流利说半年特训

  • 阅读好玩的书

  • 与妹子记录娃相关

能力提升目标：

  • redhat 中级证书

  • python 数据分析及自动化运维能力提升

  • 写作能力提升，思考金字塔原理、讲故事、写作技法、排版相关。

  • 演说能力提升。。。

计划没有变化快，唯一不变的就是变化本身，但是我们还是要制定计划。短期目标称为计划，长期目标叫战略。像我的偶像费曼、王小波、富兰克林等致敬，他们是我的灯塔，时常眺望才不至于迷失方向。

《穿越百年中东》读书笔记

发表于 2018-12-25 更新于 2022-06-29

最近对历史兴趣越来越大，看《美国种族简史》同时看了此书。郭建龙的书都写得很好。

梗概

作者到黎巴嫩旅行，在中东的信息中心，通过室友交流和实地感受理解中东的国家、宗教。
一切破碎都源于第一次世界大战奥斯曼帝国战败被瓜分。分别行成土耳其、埃及、伊拉克、叙利亚、科威特、沙特阿拉伯、伊朗、以色列、巴勒斯坦。
民族：波斯人、闪米特人、阿拉伯人、库尔德人、突厥、波斯人。
宗教：拜火教、犹太教、伊斯兰教、基督教、天主教、东正教。

  • 土耳其

土耳其是奥斯曼帝国解体后残留。突厥人，信奉伊斯兰教，什叶派。经过强势且睿智的军队统一后，建立了稳定的民族共和国。

  • 沙特

国王的国家，由沙特家族控制的君主立宪国家，伊斯兰教，逊尼派。

  • 以色列

犹太复国主义创建的国家，赶走原巴勒斯坦人。

  • 埃及

阿拉伯人，军政府统治 25 年后闹革命，穆斯林兄弟会民主政权上台后一年又被军队赶下来，迎来新的军政府。

  • 叙利亚

战火中左右夹击。西边以色列和巴勒斯坦打得热火朝天，东边伊拉克内战。

感想

破碎源于人为的界限划分，不考虑民族、宗教。国家形式社会主义、君主立宪、军政府都有。

《优衣库与柳井正》读书笔记

发表于 2018-12-03 更新于 2022-06-29

观察生活，从深入了解每个谈资名词背后开始。优衣库成了热门词，其背后时一个怎样的故事？

梗概

山口市的红顶商人柳井等是柳井正的父亲。柳家一直在山口市做服装生意，柳井等从兄弟那里学习，开始西服销售生意。柳家一个兄弟从政，很有势力。柳井等和三口组成立公司开始涉足建筑业，获利丰厚，服装生意任由儿子柳井正打点。
柳井正早稻田大学毕业，百货公司实习 9 个月，迷茫，找不到生活的意义，开始环球旅行。
一直看书，看到 GAP、Zara 销售模式，组建自己的迅销公司。无年龄、无性别、大批量、少品类、爆款为核心。高科技、无品牌、低价物美为市场营销点。
店铺管理授权负责，开店位置重要，和其他快时尚品牌同一区域。
不断开店、扩张市场，成为日本首富。

感想

公司老大就是全栈，从融资、生产、采购、库存、销售、回款，所有流程都要通关。
日本人特别看重勤奋，是上层阶层给普通民众制造的幻觉。看穿真相的人接受不了现实，大多成了御宅族。
红顶商人到首富，阶层已经固化。
小米、必要这些国内快时尚品牌服饰，竞争力很强。如果价格与优衣库看齐。手机成本和质量都能控制，更何况服装呢？
