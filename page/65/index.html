重新安装ubuntu系统

发表于 2017-02-13 更新于 2017-06-16

最初硬盘规划时，ubuntu系统只占100G硬盘空间，在使用4个月左右，系统空间开始不足，考虑重新调整分区或安装系统。以前分区4个主分区已用完，调整分区变得比较麻烦，最终决定格式化硬盘重新分区安装系统。

系统规划

硬盘320G
第一主分区50G，用于安装ubuntu操作系统，挂在/目录。
第二分区为扩展分区，占用230G，用于创建swap 8G,/home。
剩余50G空间待规划，以后可安装其他系统。

http://www.alonely.com.cn/Ubuntu/20110511/1058.html

系统恢复

ubuntu自带备份系统软件，在格式化硬盘前执行一遍备份并将备份文件拷贝到移动硬盘。待新系统安装好后，使用备份软件的恢复功能，即可恢复完大部分配置和文档，但软件是要重新安装的。

软件部署

常用的软件有输入法、secureCRT、迅雷、atom、firefox-flash、hexo

新方法计算生命

发表于 2017-01-19 更新于 2017-06-16

如果我能活80岁，从1989年3月到2069年3月，我在这世界上能存在共计29220天。截至今天，我已使用10185天，花费完生命的1/3，我获得了什么？

一想到前面的问题，难免心里一惊。如果把单位天转换为一元钱，我只有将近3万块可以花，现在也花了1万多，还没买到个啥，我该是一个什么感受？难怪孔子有“逝者如斯夫，不舍昼夜”的感慨，仔细一盘算，我也有了类似感觉。我们对时间很容易忘记，在慌忙着处理各种生活和工作的事情时，任凭时间流逝。我们无法拽住时间让它减慢，我伸手在空中抓，难以感受到空气的流动，我伸手在水里、在沙子里抓还能感受水和沙子的流逝。这也是我写博客、写每日回顾的原因，我要让文字成为我感受时间流逝的介质，让这些记录告知我时间正在流逝，我不能蹉跎岁月。

可否有一个屏幕，让我在每天出门和回家时，告知我生命余额。可否有一个机器，记录我的一天时间花费清单，在扣减余额时打印出来。看到余额减一，日清单内容时，我应该可以更具体的感受时间了。日清单的内容，目前能自动记录的有运动量等生命体征，还应有活动范围和活动内容。记录维度尽可能丰富，打印或显示维度根据个人偏好和时间阶段选择。类似于在网络中再造一个自我，并通过现实的显示和纸张和网络中的自我互动，共同养成。

每日博客或每日回顾内容可以多样化，昨天在公交车上回顾《论语十则》曾子曰：“吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？”，就是回顾的实践方法。转换为自己的内容：”今天做的事让我向目标（有钱、有趣）靠近了么？提升了我实现目标的技能（写作、写程序）等级了么？“等问题，目前只想到这两个问题，后面还可以增加，但大脑缓存有限，问题不超过3个。胡适的《赠与今年毕业的大学生》中讲到，毕业后重要带几个问题研究研究，就是自省的道理。阮一峰博文中提到”不要别人点什么菜，就做什么菜“，在每日回顾中也可以想”今天我对别人点的菜，有分析过么？“。采铜《精进》讲要做高价值长半衰期事情，”我今天做的事情是什么类型的，价值如何，半衰期多长？“。

吴军硅谷来信《向死而生》讲到过”活着总要做些事情，但是考虑到最终要死掉，并没有时间什么事情都做。我的想法常常和别人不同，我是倒着思考该做什么事情的，或则说，向死而生。我并不奢望活得比我父亲长很多，因此很容易算清楚这辈子还有多少天可以做事情。然后我把要做的事情，从最重要的开始，列一个单子，就尽量开始做了。” 他的2016年总结，30%-40%做投资，100个学时讲课，剩下时间写书两本，开专栏一个，学习两门课程。可见投资赚钱，是他最重要的事情。这应该是我写这篇博客的目的，向死而生。

王阳明心学初探

发表于 2017-01-13 更新于 2017-06-16

无善无恶心之体，有善有恶意之动，知善知恶是良知，为善去恶是格物。

看了罗辑思维204期，赶紧趁热打铁写点节目概述和感受。

讲述逻辑

道歉，大自然厌恶真空，为了自己的商业成功，以后一定坚持做好这个视频节目。
定讲王阳明的调子，个人学习王阳明的心得。粗略认识王阳明，明代中期的思想家、大官、伯爵。
加官进爵，文人军功封爵不易，王阳明做到了，厉害。#这段是废话。
王阳明心学不是成功学，是儒学的一个分支，儒学随时代变迁很复杂。
看儒学的一个视角：他的思想和谁对话？儒学思想做为皇权的解毒剂，一直在和皇权对话，这是儒学不变的。
看儒学另一个视角：儒学的目标是塑造一个君子人格。
儒学还有一个自以为是的目标，通过君子人格获得世俗成功（大官、有钱）。

两个目标：君子人格和世俗成功，针对不同人准备的不同回答。君王问什么是儒学，回答时就需要让答案显得高尚一些，要回答”儒学的目的是做一个君子品格的人，忠君报国、舍己为人、刚正不阿都是君子品格内涵“。帝王听了想，“你们一心帮我管理社稷，还不要我的权力和钱财”，很高兴，放心让他们帮忙了。孩子问家长什么是儒学，家长就要回答“你想升官发财有漂亮老婆吗？”，那就学儒学。
成功有很多说辞上的原因，都是在成功之后才说的。马云成功了，所以他说成功靠”相信我说的”。马化腾成功了，所以他说“成功源于民主讨论”。谁知道马云当年买过多少假货，马化腾当年占了多少电信局资源玩站点，这些都不重要。
