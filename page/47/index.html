《夹边沟记事》读后感

发表于 2019-09-26 更新于 2022-06-29

《夹边沟记事》读后感
最近重温阮一峰的博客，发现书单中很推荐这本书，就下载电子版，3 天看完。
博客中有一些引人思考的短句，如

算法刻画的你，才是真正的你 —你的鞋子都比你聪明

都值得记录和思考。

夹边沟是沙漠边上的人工开辟的农场，开荒人主要是 1959 年左右的右派。最初 2000 人规模，到撤离时剩余 200 人，死亡率高。全书通过访谈夹边沟的人物，揭露当时夹边沟劳教右派的生存状态。看后主要感受：1 当前能吃饱，很幸福，2 活下来就是胜利，不管通过什么方式。 3 一切都要靠自己创造。

阿里交流有感

发表于 2019-09-24 更新于 2022-06-29

背景

公司之间进行技术和文化交流，了解别人在做什么，怎么做很重要。

内容

企业文化

使命、愿景、价值观。使命回答为什么，表明存在的意义。愿景是回答是什么，长成什么样子。价值观帮助做判断，在成长过程中的具体决策，远眺使命灯塔，近看价值观路标。
企业文化不考核，不和钱挂钩，永远只能挂在墙上。企业文化要形象具体，不要太过抽象，不然无法指导行为。
阿里参考

花名：打碎原来的自己，在阿里重新成长。提供谈资，拉近距离。

商业模式

数字化、网络协同、数据智能。
数据业务化，业务数据化。
结合阮一峰的周刊更能支持这一点。个人成长，也要进行数据分析和记录。有些记录可以通过系统、物联网设备记录，但是分析只能主动复盘。

github 很多人为了让绿色小方格子不要中断，就会尽量每天提交代码。时间一长，真的多做了不少项目。
这件事的启示就是，不管做什么事，都应该进行数据记录和数据分析。对个人来说，可以从一个业余爱好者变成专业人士；对产品来说，可以从一个普通产品变成数据产品。

一起来捉妖-从入门到放弃

发表于 2019-09-23 更新于 2022-06-29

背景

中秋在家看了几集宠物小精灵动画片，突发了培养情节，搜一搜宠物小精灵相关游戏，发现腾讯的这个，入坑。

结论

游戏严重不匹配成长两字，背离了宠物小精灵的基本精神。
小智晚到了训练师中心，获得了一个有点问题的皮卡丘，并说“我也是有些问题的”，高兴的接受皮卡丘，带着它一起冒险。
捉妖这个游戏完全忽略成长，一开始就看资质，宠物的资质不满，直接放弃，因为没有培养价值。然后就开始持续的赌博游戏，赌抓住高资质宠物，赌灵石开出高资质，赌打完神兽后抓到高资质。。。比我大学时玩的页游昆仑世界还能赌。剩下的就是定时敲鼓、擂台刷排名、无意义的日常任务。
正面意义：灵石开启需要实际的步数，可以激发人们运动，到户外去。曾经想过如果大学时有这个游戏，可以和基友一起到大学城逛逛，就有游玩的目标了。定向越野活动也可以借助手机开展，应该更好玩。
一起来捉妖-从入门到放弃

留痕是成绩还是形式？

发表于 2019-05-10 更新于 2022-06-29

抛开剂量谈疗效都是耍流氓。

问这个问题的目的是期望解答：我们要不要留痕，如果要，怎么留痕。
留痕，是做事情过程中的阶段性记录。带着手机步行，每天晚上 9 点微信运动就会通知今日步数，这就是今日步数的留痕。留痕，可以是动词，通过一系列动作，达到一个效果，用刀在树上划一刀，这个动作过程就是留痕。留痕也可以是名词，留下的痕迹。作为动词的留痕，有很多种方式可以达到留痕的效果，有容易又简单。微信计步，只需要带上手机。工作阶段性效果，可能就需要会议纪要、拜访记录，需要较为复杂的分析和记录。不过好在我们是物联网公司，有好多电子设备可以方便的让我们被动留痕，比如刷脸门禁机，不用特意去考勤机刷卡，进出公司自动考勤了，也不会忘记。我还看到有公司开发出教室视频监控系统，摄像机时时拍摄学生上课画面并自动识别分析上课是否认真，是否专注，这样老师就不用刻意观察学生的行为手工记录，考试时某个学生成绩下滑厉害，可以查一下一段时间的上课记录，如果确认是上课不专心，老师就可以比较肯定的找学生说“上课专心点，成绩才有提高，最近成绩下滑和上课不专心有很大关系哟。”如果查上课记录，学生上课很专心，那老师就要考虑其他因素：考试时状态不好？突然家里变故等等。

不要太把留痕当作成绩，也不要太让留痕流于形式。我们要重视留痕，作为一种总结过去的手段，必须留痕。当把留下的痕迹当作一种绩效考核是，容易把痕迹当成绩，这也是不好的。
在不同场合，持不同观点。在太注重留痕的环境中，如什么都记录，什么都签字，我们要讲留痕不是成绩，要拿出真正有价值的效果，才是成绩。在不注重留痕的环境中，如各种过程都不记录，事情成了，有成果；事情没成，啥都没留下。这种情况就需要强调留痕是成绩，留下的痕迹作为成功经验或是反思教材，有大用处。
