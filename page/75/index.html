关于TI5的日记

发表于 2015-08-12 更新于 2017-06-16

每日一记
今天下了个早班，回到宿舍就开始看TI5决赛，看完之后有兴奋也有不安。兴奋是EG与CECD酣畅淋漓的对决，展现了个人超高的游戏艺术和团队默契。不安是想到下班后的大好时光，浪费在了和自己无关的游戏上。吃完饭回来反思这一活动，其实也没有那么堕落。

关于dota，从了解到参与再到今天只做观众，还是有不少回忆。大学是先是观看班上同学玩，看起劲后就发展同样处于入门水平的同学组成菜鸟队。菜鸟队在平台上各种被虐，从队友间相互指责抱怨到增进理解并互开玩笑，虽然被虐，但二逼欢乐多。后来毕业了，队友没了就很少玩了。没有朋友间的嬉笑怒骂，游戏只剩下寂寞了。在后来工作了开始纯看别人游戏视频，从海涛到piss，从09到情书在他们的视频中，听段子，聊感情，谈人生。时不时也关注WCG，TI这样的比赛。知道了EHOME，LGD，DK

对于自己的思考

发表于 2015-08-10 更新于 2017-06-16

在15天的假期中，大多数时间都在看手机，刷微信朋友圈看同事们出去旅游的照片，看qq空间同学们的最新状态，看知呼日报稀奇而又长知识的文章，看36kr他们的创业时代。虽然是自己的假期，但感觉自己在假期中都当了别人的观众，什么时候才能成为主角？一直缺乏勇气对自己解剖，那今天就来个开肠破肚吧。我找了个这么安静的角落，能看见的应该不多吧。
对自己解剖首先是要划分区域，是横切还是竖切。一刀两段，把自己横切了，一半工作一半生活。
在昨天饭后和女友闲聊时，抱怨女友工作时间和周期太长，执行686模式（早上6点起床，晚上8点到家，每周工作6天），都没有什么时间二人世界，太惨无人道了。反过来她问我：你对你自己的工作怎么看呢？顿时我就闷了，发现我没有认真审视过自己做了4年的工作。就算有些时候思考了发些牢骚，也就还是这么得过且过的混过来了。想想当年上学时夕阳下的奔跑，现在过的真簑。 关于工作可以问4个问题:我有什么？我想要什么？我能做什么？我要怎样实现自我的人生价值？我有3年运维经验，我要做有存在感的而不是打杂的工作，要提高生活品质。我能处理各种计算机问题、布线施工、管理维护信息系统。我不清楚要怎样实现自我的人生价值，只知道像目前这样在大单位打杂，没有积累技术能力无法实现人生价值。进半年的具体目标，我要成为前端开发者，一来可往开发深入发展，二来市场需求量大，收入可观。
关于生活，我要生活中是不是发生有趣的事情。衡量生命的长度不是那些让你畅快呼吸的时候，而是那些令你窒息的瞬间。我要平静的生活，也要清风吹过时的涟漪。近半年生活目标：住进新家，坚持跑步健身，熟练吉他，女友变老婆。

今天我决定写博客

发表于 2015-08-10 更新于 2017-06-16

一直都想写一个博客， 还曾在新浪云上部署过wordpress，但由于每次写文章都要访问网站后台-登录-选择新建文章-写文章-调整格式-发布，在这一些列固定动作之后，写文章的心情都没了就一直没有更新。今天在月光博客上看见介绍markdown的工具，其中好些都试用过比如马克飞象、简书、作业部落。试用后发现多数都不能离线使用的导出为pdf，就没经常使用。今天发现这个farbox博客工具看简介正好克服以前试用工具的不足，而且还能自建博客，真是太好了。
每当读到到csdn、博客园等博客社区大牛和独立博客达人的技术文章解开技术疑惑时，有感谢分享更有感慨：他们学习实践了那么多技术并通过博客记录分享出来，而我工作了4年，记录的问题和解决方案零零散散的飘落的硬盘的不知名角落。数据丢失风险和不能方便回顾，这些信息又有什么用呢？于是在心里默念：我要坚持写博客，记录技术也记录生活。
社交网络发达的今天，记录生活工作的平台很多，但我不想在上面记录。社交网络如qq、微信、csdn等上面熟人太多，不好在上面无病呻吟生活和记录简单的技术问题，影响我高大威猛的形象。我只想找一个角落安静的自我剖析，找个空间记录我在工作上的摸爬滚打。
这个farbox平台看上去是一个不错的地方，首先支持markdown格式编辑，不用再为格式问题而使自己思路中断。其次是文章以个人站点方式保存在个人网络存储里，数据安全信有保证。另外可以方便的切换不同站点也给我带来了惊喜。
