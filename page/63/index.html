Ubuntu-wine折腾记

发表于 2016-11-23

事件起因于想在ubuntu系统上连接MS sqlserver 数据库。之前在windows环境使用了Navicat的产品，连接MS server很方便。昨天经过一番查询，发现可以在ubuntu上运行Navicate prumu连接各种类型数据库，就下载下来运行。运行前提是要有wine，就sudo apt-get install wine,安装了wine。一开始下载的是中文版，运行后窗口中文显示不出来，就下载安装了英文的。结果安装后可以连接MySQL,ProgresSQL,Oracle,SQLite,MarriaDB就是不能连接MS sqlserver。之后还找了squirrel 在java上运行的数据库连接平台，由于驱动没找到也放弃。后面使用wine安装了Navicate for sqlserver,能成功安装但无法运行。终于放弃，在虚拟机上安装了。
