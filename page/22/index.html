《论中国》读书笔记

发表于 2022-07-04

发表于 2017-07-27
吴军在给中学生书单中，提到这本书，两天时间看完，对中国近代史有了更进一步了解。
亨利.基辛格以参与者的视角，讲述了自 1949 新中国成立到其后 40 年的中美关系史。试图解释中国人对战争与和平、国际秩序等问题的思考，以及这种思考与美国人更为实用的、就事论事的思维方式的关系。不同的视角，有不同的体验。
新中国成立后，毛在莫斯科两个月，与苏联达成同盟关系。此关系维持到对越自卫反击战，彻底断裂。中国对外关系一直在美苏之间平衡，以获得自身的最大利益。
跟进苏联的大跃进运动、人民公社运动掀起意识形态的斗争。文化大革命、上山下乡，内部整风过头。领袖的去世与权利的交接产生不小动荡。但党都选择了最有利于国家和人名发展的道路。到现在我们享受了发展的成果，党的执政信心由内向外散发。发展会遇见问题，道路是正确的，只要向前不断摸索，定能实现伟大复兴。
抗美援朝，是中苏同盟的一块试金石，苏联开始信任，开始援助中国。印度边境纠纷、民间支持北越、金门炮击也是这一时期同盟关系的产物。
对越自卫反击是摸苏联老虎屁股，开始反抗苏联，于美国关系升温。
40 年的基本稳定，中国内部斗争 20 年，改革开放 20 年，终于国富民强。美国发展了 200 多年，中国的发展之路还早呢。
对个人而言，毛的时代相信国人的耐受性，无畏无惧。邓的时代引导国人创造性，发挥创造性致富，幸福生活，就是个人主题。

《从一到无穷大》读书笔记

发表于 2022-07-04

发表于 2017-07-25
吴军多次在来信中提及这本数学启蒙书，花了 1 天阅读完果然受益匪浅。
全书分为 4 部分。第一部分讲数学，讲大数，虚数。把数放在无穷大的空间，整数和奇数一样多。能否有两个数，相加为 10,相乘为 40,这个问题就引出虚数概念。两个数 5+根号-15 和 5-根号-15 这两个数就满足条件。虚数在几何意义上为实轴上数旋转 90 度的结果。
第二部分讲时间、空间与爱因斯坦。长宽高为三维坐标，再加上垂直于这个坐标的时间轴，就构成了四维空间。通过二维展现三维的类比，在三维战时四维。第四维的度量单位为光速。太阳系是扭曲的，行星按照最近的轨道运行，所以是观测到的椭圆形。
第三部分讲微观世界。从物体逐渐细分追溯到最根本，世界是由核子、电子、中微子，核力、电磁力构成。裂变是用中子轰击原子核生成同位素并释放能量。聚变是高温下氢原子在核力作用下聚变成氦原子，释放出能量过程。无序运动通过概率来描述，有序到无序的概率变化，称为熵。整个系统是一个由有序到无序的过程，成为熵增。病毒是介于生命体和有机体之间的物质。染色体是由于细胞内此类物质较易被染色而命名。隐形遗传、显性遗传、基因突变、基因交换。
第三部分讲宏观世界。我们有于视差，观察到更远的视野。宇宙撞击说、大爆炸说起源。恒心、白矮星、红巨星、超星星诞生。
整本书横跨数学、物理、生物、宇宙，极大扩展了我的知识。天空为什么是蓝色的？大气布朗运动造成大气层折射保留蓝色光。海洋反射天空的颜色，所以是蓝色。盖莫夫的科普书看来很值得度，这本书由于有较多的图表，比较适合购买纸质书。10 岁左右的小孩比较适合看。

1 Automating Installation with Kickstart

linux 系统安装过程中的人工选项会保存在/root/中，名称”anaconda-ks.cfg”,如果需要批量安装物理机，可以安装好一台后，将此文件替换为 iso 镜像中的 ks 文件，然后就不需要手动选择了。目前主要通过 consle 装刀片，没有 dhcp 这些环境，定制化 iso 比较可行。
可以用图形工具”system-config-kiskstar”和文本编辑器编写 ks 文件，带函数检查的文本编辑器“ksvalidator”.在系统安装选择界面按 tab，参数输入”ks=cdrom:/dir/file”，即可选择不同的 ks 文件。
更大规模的自动化部署物理机，需要用到 PXE 和 kickstart。cobber 是一个 linux 环境下的集成 PXE 需要的 dhcp、tftp 服务器的一个自动化部署环境。

克隆虚拟机后快速修改ip

发表于 2022-07-04

发表于 2017-07-18

问题

需要根据模板建立 26 台虚拟机，ip 地址不一样，其他完全相同。

解决方法

  1. 根据需求从模板克隆一台 centos6.5 虚拟机，修改硬件配置（CPU、内存、网卡）。
  2. 进入新创建虚拟机，修改网卡配置文件 ip 部分,包括 ip 地址、子网掩码、网关。
  3. 删除 net 文件 rm /etc/udev/role.d/70-*-net
  4. 在/root/中创建修改 ip 脚

执行方式 sh 1.sh 122,即修改 ip 地址为 192.168.38.122 并重启网络。

  1. 删除 net 文件，关机
  2. 克隆此机器多台，启动后带参数执行 1.sh 即可快速修改 ip 地址 sh 1.sh 122

总结

思考重复动作，优化重复动作执行效率
脚本文件不具有普遍适用，但还是能快速解决问题。
