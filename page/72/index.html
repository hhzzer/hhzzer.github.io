论植物大战僵尸2的沉迷系统

发表于 2016-05-02 更新于 2017-06-16

最近心血来潮，重新打开了ipad上的植物大战僵尸2，完了几天就沉迷进去了，到今天持续了25天。这游戏为什么让我持续的投入时间，有必要分析一下。
签到机制吸引人每天不要忘了上线。因为签到动作很简单，获得奖励又很丰厚，有植物、钻石、金币等，往往提醒我，就签个到，花不了多少时间，然后就开启应用。而且持续签到天数越多，获得奖励越丰厚，就不忍中途放弃签到，也是让玩家持续上线的原因。
签到完之后就想，反正也打开了，要不收一下菜，反正也就1分种的事。于是资源采集系统又让你沉迷2分钟。训练僵尸，开拓土地，定时收获勋章，这种类似于开心农场的机制让你在收完菜后就计算，还有多少个小时可以再收获多少，提醒你4、5个小时再来收获一次。毕竟资源掠夺过来，不收获就是白费力气了嘛。
资源掠夺和收完菜后，在看一下pvp系统，操控僵尸进攻别人的花园，每天成功5次可获得50钻石。这是除充钱、固定关卡和签到获得钻石的最后一个机会，而钻石又要经常用到，那就完成日常吧。10分钟又过去了。
逛完pvp系统，左右各有一个感叹号，日常任务标志。植物探索和每日挑战可获得植物碎片、装扮。除了钻石抽奖，就只能通过这两个日常刷了。后面有些关卡，没有高阶植物根本打不过去，又不想充钱抽奖获得碎片，怎么办呢?刷满日常吧，虽说有些枯燥无聊，但有利于后期发展呀。20分钟的日常挑战又过去了。
植物探险也是一个需要定时地点操作的任务，就是要时不时的把你拉回来。
相比植物大战僵尸1，植物大战僵尸2和腾讯合作发行后，越来越有国产无耻手游的影子。抽奖、集碎片、装扮、配饰，反是能增加游戏玩家在线时间的系统都上，越来越没有乐趣了。

写完此文章，我已经绑定了账号保存进度，然后计划删除这个游戏，因为这个游戏已经不能叫做植物大战僵尸了。好玩的游戏应该有美感和享受的，而不是设计各种沉迷系统骗钱。bye，我的植物大僵尸。

游戏是个怎样的世界-观后感

发表于 2016-04-28 更新于 2017-06-16

今天看了罗辑思维新一期的视频，以游戏为主题，自己很感兴趣，就认真看了一遍。

视频以吸毒后的的幻想世界与贩毒的真实世界未引子，引出电子鸦片概念、虚拟与现实到底谁好的问题，开始分析。然后开始将劝解上网少年的故事，并站在上网少年的角度，对劝解提出质疑。一 、沉迷于虚拟世界中不好。对应：好与不好关键在于人多人少，基督教开始时被认为邪教，到后来教徒数量达到超过其他教时，基督教的好坏，别人说了也不算。二、长期玩游戏有害健康。对应：人类发展都是向不健康的趋势发展，原始人到处活动锻炼有益健康，现代人挤在一堆吸汽车尾气有害健康，但社会就这么发展了。三、玩游戏没前途。对应：游戏选手和游戏解说挣大钱，游戏大有钱途。
为什么我们喜欢玩游戏？也是从提出假说开始说明的。我们玩游戏是因为游戏细节有趣。辩：俄罗斯方块没啥细节，一样风靡全球。玩游戏是因为能赢？有些游戏一开始就注定要输，比如俄罗斯方块。第三个忘了。 辩驳完假设后提出：我们喜欢玩游戏是因为游戏中给我们的行为提供实时丰富的反馈，而这个对于行为的反馈需求，是刻在基因上的。游戏中的勋章系统、数值系统、排行系统反馈我们在游戏中的过去、现在和将来。游戏的反面不是工作，而是无聊与恐惧。我们对于太容易的事情会感到无聊，对于太困难的事情感到恐惧，做介于无聊和恐惧之间的事情则会产生幸福感。游戏中恰好有很多做这种事情的机会。
未来我们会不会迁徙到虚拟世界中？不会从现实中跳进虚拟，而是慢慢迁移。人是什么？人只不过是悬挂在自己编织的意义世界中的动物。用来解释虚拟和现实的区别，感觉就像马斯洛需求金字塔一样，现实是基础，但追究的目标是虚的。

每次听罗辑思维，都觉得讲得很有条理，待我把这一期多听两遍，梳理各脉络出来，更容易记住。虽然有些观点有些牵强，但这种有条理的表达方式，是我需要认真学习的。

最后推荐书籍有《游戏改变世界》《有限与无限的游戏》

2016-04-19晨跑之后

发表于 2016-04-19 更新于 2017-06-16

好久没有早上跑步了，虽然闹钟一直没有取消。昨天又看了王小波的几遍杂文，晚上心情有些激动，就做了今天晨跑的决定。和往常一样，闹钟响了，眼睛还睁不开。摸着墙到洗手间解决并洗漱完毕，穿鞋出门。出电梯到小区道路，看了天边的鱼鳞云，今天又是个好热的天气。
跑动起来后，眼睛才完全睁开，因为要注意别撞这来往的车辆。过完十字路口，跑上路旁的人行道，才算步入正规。今天出门，手机都没带，因为带着手机就想一边听歌一边跑步，但是没有找到合适的耳机，跑步老掉下来，索性就不带了。当我们在跑步时，我们在想什么？伴随着旁边工地平地的打桩声，我开始思考这个问题。打桩机让泥土在空中飞舞，最近看的各种博客也让我的思维漂浮。我该写一篇关于最近关于最近看的文章的汇总和看法，来沉淀思维。这是在跑步时做的第一个决定。真的，只有通过写的过程，才能网住思维的蝴蝶。

在工作的间隙空闲，我就喜欢看rss。以前看科技新闻比较多，现在看个人博客较多。因为科技新闻只是你可能需要知晓的，而喜欢的博客主写的博客，则需要思考。怎么发现我喜欢的博客呢，一是看blogbar.cc ,一个个人博客的聚合，看了个人博客之后再看他的朋友链接，这样就能拉出一个大象。另外就是看更自动化的博客聚合github，看偶像的关注。看的个人博客技术方面基本上都是关于docker、node.js 、react native、javascript等偏前端的东西，因为一个人独立创造一个东西，虽然还没想好要创造什么。然后就看了博主些关于工作的一些看法，其中四火和hustlzp有几篇对工作生活态度的文章写的很好。工作，不是为了绩效、公司或其他人的看法，工作成果的第一责任人是自己。工作中获得些许金钱，但更多的财富是工作中积累的技术能力，沟通能力及对事物的一些看法。

我还看了静风说、黑客与画家、沉默的大多数。
