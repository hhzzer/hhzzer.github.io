k8s in action 服务

发表于 2021-03-04 更新于 2022-06-29

service，是在 pod 上层定义的资源对象，为了对集群内外部提供一个统一，稳定的访问入口。
为什么需要呢？
pod 短暂，ip 易变，服务使得水平伸缩不影响服务提供。

在内部创建和测试服务


三种方式从集群内部测试服务

  1. 创建一个 pod，从 pod 内部请求服务 cluster-ip，然后通过 pod 日志查看服务响应
  2. 使用 ssh 登陆 k8s 集群一个 node，使用 curl 命令
  3. 使用 kubectl exec 命令在集群已存在 pod 中执行 curl 命令

同一个服务可暴露多个端口，在 spec.ports 中写就行

如果在 pod 中给端口命名了，spec.containers.ports.name=http ,spec.containers.ports.containerPort=8080,在 service 中 targetPort 就可以直接用 port 的名字 。

pod 中的 client，或则 node 中的 curl 怎么发现服务呢，怎么通过 cluster-ip 访问到服务的呢？

  • 通过环境变量 kubectl exec podname env
  • 通过 DNS
  • 通过 FQDN

连接集群外部的服务

endpoint 是暴露一个服务的 ip 地址和端口的列表。endpoint 是 service 之下的对象。2 步将外部服务引入集群内部，1 创建一个 endpoint，2 创建一个没有 pod 选择器的服务。选择第一步创建的 endpoint。配对通过 metadata 中的 name 匹配。


还可以不创建 endpoint，直接为 service 指定 ExternalName，pod 可以通过 service.name 连接到外部服务。起作用的原因是 dns 添加了 CNAME 记录。CNAEME 只能指向域名，不能指向 ip。

将服务暴露给外部

  • 服务的类型设置为 NodePort
  • 服务的类型设置为 LoadBalance
  • 创建一个 Ingress 资源 ，通过一个 ip 地址公开多个服务，运行在 http 层。

NodePort 示例

内部 pod 可以通过 clusterip：80，访问。集群外部可以通过 NodeIP：30123 访问，如果有多个 Node，每个 node 的 30123 端口都可以。

负载均衡器暴露服务

ingress 定义：进入或进入的行为。
为什么需要 ingress
每个 loadbalance 服务都需要自己的负载均衡器，以及独有的公网 ip 地址，而 Ingress 只需要一个公网 ip，就能为许多服务提供访问。

云原生时代的运维新归宿

发表于 2021-03-01 更新于 2022-06-29
  1. 背景：云原生技术快速发展和普及，已经进入云原生时代。
  2. 传统运维困局及原因。责任多，传承难。
  3. 云原生时代运维特征。基础设施即代码；运维特性前移到开发阶段聚焦基础设施层；基础设施可靠性和运维自动化。
  4. 如何在云原生时代运维？用软件工程思想去完成运维自动化。要进行云原生应用交付，需要云原生自动化运维平台。
  5. 运维转型：熟练掌握容器和 k8s，掌握 go 语言，关注 devops，云原生社区，学习使用开源工具解决问题。
  6. 云原生运维价值。助力业务更敏捷可靠。
  7. 生产力担当？好基础设施有利于创建好产品，快速迭代增加企业竞争力，技术开放降低生产成本，中台模式提升效率，快速试错、反馈增活力。

发给妹子的文章摘要汇总-8

发表于 2021-02-15 更新于 2022-06-29

基本上，如果想成为好作家，就必须做三件事：多多阅读，仔细深刻的倾听，以及多多地写。同时，不要想太多，只要长驱直入文字、声音和各种知觉的核心，并且让你地笔在纸上写个不停。
—《写出我心》

我这么试过，看了文学理论后，发现这是意识流。

教父里的人生观：
第一步要努力实现自我价值
第二步要全力照顾好家人
第三步要尽可能帮助善良地人
第四步为族群发声
第五步为国家争荣誉。
事实上作为男人，前两步成功，人生已算得上圆满，做到第三步堪称伟大，而随意颠倒次序的那些人，一般不值得信任。这也是古人的“修身齐家治国平天下”。所以，在你想要准备开始“平天下”的时候，也得先想想，自己的生活有没有过好，家人照顾好了么。。。

在聊职场八卦，不同工资女生睡觉前都在想什么时，提示要多想自己和周围的人，少操拯救地球的心。

生活方式以及是否追求财务独立的决定权都在于我们自己。我们有两种选择：一是随大流（成为市场营销者和销售人员的目标）；二是默默地追求我们的经济自由。
我们中 80%的人都认为伴侣在通往财富成功的道路上扮演非常重要的角色
经济独立属于那些愿意分配时间、金钱、精力和认知资源以实现财务目标的人。
《财富自由》

妹子，你愿意和我选择第二条路前行吗？

因为劳动收入转瞬即逝。那些获得财富自由的人靠的是储蓄存款和投入资本产生的非劳动收入。
收入是你今天带回家的钱，而财富确是你明天，后天以及今后每一天所拥有的钱。

我们也要增加非劳动收入。或者让劳动成果持续产生收入。

  1. 工作是一场马拉松，而不是短跑。
  2. 尊重人，而不是头衔。
  3. 工作从白板开始，而不是从键盘。
  4. 产出的是价值，而不是代码。
  5. 热爱生活，而不是工作。
  6. 学会享受过程。
