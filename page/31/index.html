在线-读书笔记

发表于 2022-06-29

发表于 2018-07-27
leader 推荐王坚博士的《在线》一书，读完写下摘要和感受。
全书分为 10 章，主要由媒体访谈内容组成。王坚博士在媒体访谈中的解答可以用 5W2H 梳理。整本书的内容和标题很相符，就是说明：在这个万物互联的时代，数据很有价值，只有在线的数据，才能发挥出数据的价值。

Why 为什么有阿里云

对标国际互联网公司，国外有亚马逊和谷歌，国内却没有规模的大公有云计算公司。GWF 和国家信息安全政策环境下，国外云公司很难进入中国。在这片蓝海领域，阿里做公有云计算，做成了类似于水电气的通用资源提供商，可以行成垄断，获利丰厚，很有商业价值。
公有云进入门槛高，阿里有技术和资金，应该可以成功。

When 什么时机做阿里云

2006 年亚马逊开始提供公有云，2008 年阿里开始做。

What 什么是阿里云

阿里云是资源公司。提供计算能力、存储能力、网络能力。接上就能用，而且价格便宜。

Which

Where 阿里云的范围

和电能与电器类比，阿里只做发电机和电力传输网络，负责把资源运送到普通人手里。这些计算、存储、网络能力能怎么用，在什么场景下发挥价值，就是电器厂家的价值所在。

How 怎么做的

从 0 做起。做分部署存储系统-盘古，做分布式资源调度系统-女娲，做资源诊断系统-华佗，做钟馗。。。形成一个由 5000 台物理服务器组成的网络操作系统-飞天。

How Much 花费

前期基础技术研发费用高，一旦研发成功，收益丰富。资源用户可以很便宜的获取资源。
计算是在线的，不是私有网络的几个机架的服务器。
数据比功能重要，功能很快就能通过技术被模仿，用户需要的是数据支撑的服务。
做 app 是在别人的花园里弄盆栽,所以要做 yunos。

yuque-hexo 安装路径中的 config.js 文件，util\imageBeds\qiniu.js，注释掉 token,写上自己的。

过程

git clone 下载项目代码，在 node 中逐个调用函数测试。清除下载后，需要删除 hexo-yuque/source 目录才会重新生成 md 文件。

浪潮BMC登录-加密处理

发表于 2022-06-21 更新于 2022-06-29

背景

使用 python 登录浪潮服务器的 BMC 登录，发现传递的用户名密码经过了处理，需要在 python 程序中也做处理。

分析

结论：浪潮 BMC 登录加密用户名密码信息有 2 中方式，不同版本用不同的加密方式。模拟登录对所有版本的 BMC，只要 post 的加密类型（encrypt_flag） 与密文能对应，服务端都能正确认证成功。为模拟登录方便，python 程序中都使用 encrypt_ctrl=1 ，异或编码。

分析 ajax 传递登录信息的页面，发下有两种类型的密文。
第一种密文

观察页面的表单提交执行的函数为 doValidate 中的 doLogin 函数。doLogin 函数中定义了 ajax 传递的内容，username 使用了 bf.base64Encode()对表单中的内容进行加密和编码。加密函数 bf 由 Blowfish 创建。


在浏览器 console 中手动运行 bf.base64Encode，输出与 playload 一致。

第二种密文

观察表单页面，也是关联到 doValidate。在 doValidate 函数中，ajax 传递的值是 encrypt_username.
encrypt_username 根据 encrypt_ctrl 值不同，有 2 中处理方式。encrypt_ctrl = 1，使用 encryStr 函数加密。encrypt_ctrl=2，使用 bf 函数加密编码。

encryStr 函数内容.处理过程为字符串的 unicode 与 127 取异或，然后转换成 16 进制，用-连接起来。
