vmware分布式存储vsan规划与搭建

发表于 2022-07-04

发表于 2017-11-17
公司要部署一套 vsan 存储，选择在 RH2288H v3 上。记录下思考和实践过程。

基础概念

通过高速以太网络，连接服务器上的本地存储，组成一个带冗余，高读写速度的存储网络。
使用 4 层协议。使用 ssd 缓存加速 io 读写。使用软件层创建多副本和见证，提供冗余。关键技术：软件通过磁盘控制器直接读写磁盘。软件相当于功能更复杂的 raid。
磁盘控制器能力和磁盘能力是 vsan 性能的关键。esxi 系统和管理的数据采用独立的磁盘控制器。读写数据盘的磁盘控制器读写深度>255。磁盘和控制器之间不能加控制器扩展卡，变相使控制器能接更多的磁盘。hhd 磁盘选择 sas 或 nl_sata 接口，速度 3-6Gb/s，转速 15k-7.5k 之间，ssd 选择读写次数多的。网卡选择高性能的 10gb。vsan 分全 ssd 和混合，混合居多。

博客支持快速插入图片

发表于 2022-07-04

发表于 2017-11-16
今天修复了博客的 tags 和链接，继续优化添加图片。

问题

博客插入图片不方便，结合七牛云存储 shell，快速插入图片

实现方法

1 使用七牛作为图床，注册 7 牛账号，新建一个存储区域。
2 下载七牛命令行工具 qshell，方面通过脚本上传。
3 编写 bash，bash 中结合 qshell，做到上传后讲链接拷贝到剪切板。
4 将图片保存到本地目录，在 shell 执行 up-img file-name,剪切板会获得图片 url。
5 写文章时使用格式，插入文章
up-img

职业发展总结

发表于 2022-07-04

发表于 2017-11-15
专栏半年总结第二篇为职业发展话题总结。
分为三部分总结，1）职业的底层概念 2）个人能力发展 3）选择好外部环境。
职业就是满足他人需求获取收益的一种社会交换，职业发展就是提高交换效率。对内提高能力，对外选好战场。

底层逻辑

职业是通过满足他人需求获取收益的一种社会交换。职业发展就是让交换越来越高效、高价值的动作。
个人有能力，有赚钱或发展的需求，组织有要做的事，也能提供回报，所以个人在组织中进行社会交换效能高。如何实操就是先跟后带，先满足组织需求，在满足需求之后满足自己发展需求。心法：把自己当做一家企业来经营。开餐馆市场调研-满足哪些人群的什么需求，选址，菜品等等。
职业发展最终状态为自由职业态，分位三个层面：1 不需要你非得为某个公司工作的自由（能力提升后解锁）2）不需要为钱工作后的自由（睡后收入支撑生活成本后解锁）3）有能力和资源做自己想做的事的自由（积累能力和资源后解锁，选择范围决定自由度）。需要产品、媒体、运营三种元能力。

对内修炼

能力三环：知识、技能、才干。通过快速学习知识，组装技能和才干，获的跨行业的竞争力。
新木桶原理，要投资自己的优势，与人合作。

对外选择

企业结构为三层 8 区金字塔，向上、向内、左右、向外 4 中发展方向。
行业视角。行业发展分位曙光、朝阳、成熟、夕阳 4 个时期，要借助行业发展的扶梯上行。最好进入行业的时期是朝阳的前中后期。起步者的机会：二级市场、细分市场、三四线城市。
求职策略。根据雇主思维去求职。内部员工、作品与实习、内推、面试、简历。

自我管理和高效能

发表于 2022-07-04

发表于 2017-11-14
昨天胡乱总结，今天认真听了，还是老师总结得好。
所有行动或目标期待都源于人的能动性，所以一开始要认识自己。选择比努力更重要。

  1. 断舍离，清晰判断什么事是重要的。
    断，断掉不需要的东西；舍，舍去多余的东西；离，离开对物质的依恋。通过对东西的断舍离，找到情绪告诉我最重要的东西。通过对事情的断舍离，找到价值观告诉我的最重要的事情。真心认为重要，才有做事情的动力。断舍离是一个观察自己价值观并让价值观逐渐清晰的方式。
  2. 三件事和结果思维，把事做得有价值。
    每天记录要达成的三个结果并加上时间累计标记，发布到公开或半公开的地方，利用为自己保全面子的冲动去完成结果。有价值的事是有客户价值的事，客户愿意掏钱或掏时间和你交换。让结果更有价值的可从显性化、可交付、可复制三个角度努力。
  3. 心里比对和如果那么，提前制定做事的策略。
    做事之前的策略很重要，凡事预则立，不预则废。心里比对是把事情最好的结果和最差的结果对比，让我们避免盲目乐观而轻视要做的事。也让我们进一步筛选事情是不是头脑发热的冲动。
    如果那么就是在做事情之前想好可能发生的意外情况并为意外情况做好预案。等到意外发生时再做选择，大脑容易选择懒惰。多做如果那么规划容易把事做长久。
  4. 碎片化时间管理的加减乘除方法。
    碎片化时间不可避免，用合适的姿势拥抱是最佳策略。加，增加碎片化时间的价值。1 分钟做舒展运动，5 分钟做策划，每个碎片都应有产出。减，减少事件切换。将同类型事情累在一起做，打电话、回邮件批量执行。乘，多人有效的事情一起做，比如结队编程。学习效能金字塔指出调动感官越多，学习效率越高。独学而无友，则孤陋而寡闻。除，把大目标打碎分解，消除目标恐惧，阶段性获得正反馈有利于事情推进。
  5. 有时间也要有状态，精力需要管理。
    精力有限，有浓度曲线，在精力高时做高消耗精力的事，低精力做低精力消耗的事。精力可恢复，可锻炼。平时使用不要过度消耗，做了极限尝试后要花足够时间恢复。
