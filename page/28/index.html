ubuntu python selenium 调用firefox找不到驱动解决

发表于 2022-07-04

发表于 2018-03-31
学习 python-web TDD 时遇到 selenium 调用 firefox 问题，记录一下。

解决方法
从 selenium 3.x 开始，selunium 调用 firefox 驱动名称改为’geckodriver’.
firefox 47 以上版本需要下载三方驱动放在系统环境变量中。
查看当前 selenium 和 firefox 版本信息，发现符合上述两个条件。

pip list
#result
Django (1.7)
pip (9.0.3)
pytz (2018.3)
selenium (3.11.0)
setuptools (38.5.2)
wheel (0.30.0)
apt list firefox
#result
firefox/xenial-security,now 59.0.2+build1-0ubuntu0.16.04.1 amd64 [installed,automatic]
下载并移动到环境变量路径中.到https://github.com/mozilla/geckodriver/releases
上找到对应的版本，下载。

tar -xzvf geckodriver-v0.20.0-linux64.tar.gz
sudo mv geckodriver /usr/bin/
重启 bash，进入 virtual 后，测试故障排除。

##附加收获 锁定 firefox 版本
查看可用版本

apt-cache show firefox | grep Version
结果

Version: 47.0+build3-0ubuntu0.16.04.1
Version: 45.0.2+build1-0ubuntu1
install

sudo apt-get install firefox=45.0.2+build1-0ubuntu1
keep this version

sudo apt-get install firefox=45.0.2+build1-0ubuntu1
解除锁定，升级版本

sudo apt-mark unhold firefox
sudo apt-get upgrade firefox

sslocal失败解决方法

发表于 2022-07-04

好久不锻炼，都不知道动作了。

ss_config

使用方法 sslocal -c ss_config
使用会报错“AttributeError: /usr/lib/x86_64-linux-gnu/libcrypto.so.1.1: undefined symbol: EVP_CIPHER_CTX_cleanup”，查询后是由于 openssl 版本升级，修改了以前的函数名称，而 ss 还使用了旧函数名。解决方法是修改旧函数名为新名称

邓小平传-读书笔记

发表于 2022-07-04

发表于 2018-06-09

好久没更新博客了。一是没有经常使用笔记本，写博客只能在私用笔记上，就发文较少。二是用了一段时间蚂蚁笔记写博客，还充值 50 购买一年会员，以为会在上面常驻，结果不了了之。三是记录工作文档都在语雀上。

在网易蜗牛阅读上，花了 3 周时间把邓小平传看完，记录下。
邓小平 1904 年出生在四川广安，有亲戚做生意，资助他上学。清政府洋务运动和法国大革命后缺劳动力，通过层层考试 16 岁去了法国勤工俭学。到法国没上几天学，就去工厂上班糊口。加入法国的华人共产党，给书记周恩来做助理印党内刊物。法国混了一段时间后又去莫斯科，和冯玉祥的儿子一起上学。国共第一次合作时回国参加共产党运动。南昌起义后在广西发动百色起义。国共合作分裂后随红军长征到延安。到延安与第二任妻子结婚，毛泽东等出席。
解放战争时刘邓大军，邓小平为政委。文革时期下放到江西鹰潭。文化大革命末期周恩来招进北京，准备重用。对文化大革命的态度导致再次下台修养。1976 年周毛逝去，与华国锋一起领导中国。
与华国锋共事时对两个凡事有不同看法，出现分歧。十一届三中全会正式独揽大权，开始改革开放。三个特区深圳、珠海、厦门在开放中迅速发展。
对外外交强硬坚毅。解决香港问题，坚定台湾立场。打击越南和苏联，和美国日本亲密接触。
1992 年南方专列，加速改革开放。
看完后对领导人有了多角度的认识。胡锦涛处理西藏问题厉害，江泽民处理学生问题当主席。曾庆红是延安幼儿园长家的，关系广。薄一波为财政部长，习仲勋广州书记经济建设能力强。朱镕基与世界银行专家一起诊断中国经济，经济能力好。赵紫阳，胡耀邦对言论宽容引发不稳定下台。万里处理铁路运动，李鹏为周恩来养子 1989 年维稳功劳大。叶剑英为上海富家子弟，不热衷政治，大家都好。
国家发展哪里，哪里就能发展，读懂政策很重要。深圳特区就是例子。
二胎政策开放就是增加房地产需求，供不应求，房价就要涨。
后知后觉，要总结经验教训。
