文件共享方法

发表于 2016-11-21 更新于 2017-06-16

最近换了操作系统使用Ubuntu后，经常要与其他同事共享文件。本来内网有文件共享服务器，但使用文件共享服务器需要先上传到服务器，或别人上传了再下载下来。一是步骤多，二是在安全问题，大家都能访问。查看资料后发现两个python文件，就能基本满足要求。

1 python SimpleHTTPServer
SimpleHTTPServer是安装python后自带的一个文件，安装python后就可执行，无需安装其他库。在需要共享的目录下执行python -m SimpleHTTPServer $port_num后，别人访问你主机对应的端口，即可浏览并下载文件。

2 python droopy
实现自己文件共享后，需要一个途径收集别人的文件，别人不一定能将文件生成下载链接，这就需要我生成一个上传链接。需要传文件给我的人通过链接，就能上传到我的电脑指定目录。百度发现了droopy,下载地址：http://stackp.online.fr/wp-content/uploads/droopy 。该文件可以生成一个上传链接，传递单个文件。
python ~/bin/droopy -m "Hi, it's me Bob. You can send me a file." -p ~/avatar.png

3 本想着考虑共享文件的安全性，生成带密码和过期时间的链接，未果。先这样用也很方便。
