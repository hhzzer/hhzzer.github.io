ubuntu无法执行passwd命令排查

发表于 2017-06-16

背景

使用模板创建ubuntu14.04系统时，修改root密码报错

解决方法

  1. 查看auth.log,发现是缺包

  2. 安装缺失的包apt-get install libpam-cracklib

  3. 验证执行passwd,不报错。

    参考

    https://goinggnu.wordpress.com/2015/06/04/solution-for-passwd-module-is-unknown-issue-in-ubuntu/
