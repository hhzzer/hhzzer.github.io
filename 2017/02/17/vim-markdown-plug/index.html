ubuntu系统使用vim编辑预览markdown文件

发表于 2017-02-17 更新于 2017-06-16

使用atom一段时间后，发现启动速度和命令行查看不习惯，尝试一下vim+markdown插件如何？

安装

  1. 安装vim插件管理工具Vundle

git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

修改vimrc,如下内容添加在配置文件末尾sudo /etc/vim/vimrc

打开vimsudo vim，在命令模式执行PluginInstall，即可安装成功。

  1. 安装其他插件
    待安装插件写在配置文件Plugin 'VundleVim/vundle.vim'之后，并在vim命令模式执行PluginInstall即可。
