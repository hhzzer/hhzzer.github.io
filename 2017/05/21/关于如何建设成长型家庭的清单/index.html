关于如何建设成长型家庭的清单

发表于 2017-05-21 更新于 2017-06-16

最近罗辑思维早上60s改版了，语音消息变成图文推送。第一条是各类型清单，今天就写一下建设成长型家庭清单的读后感。

这张清单实质是优化生命环境的一系列建议，生命环境包括这一生要接触的东西，包括独处时刻、家庭时间和工作时间。清单只针对家庭时间来提出优化建议。

时代变了。以前通过家庭分工抵抗风险，现在靠不断学习抵抗甩你到社会边缘的离心力。家庭要建设为成长型的，才能不被甩到边缘。

能否建设成长型家庭，要看家庭成员是否具有成长性。考察方式是看最近一个月有没新的知识获取渠道，最近一个月没有，一直也不会有。选择大于努力。相比原始资本，要更看中一个人的成长性。

有了成长型的成员，成员还需要有独立的自我目标，在实现自我目标的过程中培养自己的合作者姿态，带到家庭中效果才好。个人目标的自我推动力大于家庭目标。

可以尝试的案例是逛公园分目标，交通路线规划、票价走势分析、植物识别。bbc科学纪录片老少皆宜，开脑洞、增谈资、主动查证。

小孩孤独可以家庭结伴成长。选择标准依据学习能动性而不是旧情。

结合最近听吴军硅谷来信，人生最重要的投资（上、下），男生择偶选择聪明的，女生择偶选择对自己好的。两个人结合是相互包容理解的结果，优点缺点都要接受，接受一段时间可能没啥问题，一直这样问题就大。所以要不断优化自己，别指望一直会被包容。
