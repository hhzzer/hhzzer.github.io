跟踪经济的思考

发表于 2017-05-25 更新于 2017-06-16

物联网的本质是跟踪，充分利用工作的环境，理解物联网，挣大钱。

吴军在《硅谷来信》中讲了未来经济的特点共享经济、跟踪经济、合作经济、众筹经济。跟踪经济分两天讲，重点来理解下。

物联网的到来，让我们对人和物的跟踪方便便宜。传感器收集信息方便。

机器智能使得分析跟踪产生的大量数据，抽象出模型，产生价值花费减少。

跟踪的目的是更详细的了解跟踪对象，对象包括人和物品。

对物品的深入了解有助于物品的生产者、销售者和消费者。制造者了解物品的生产过程、流通过程和消费用户，能获得直接的反馈信息，用于改善设计、质量。销售者获取物品上下游信息历史数据有助于精准营销。消费者了解物品渠道和生产信息能减少购买假货、劣质品。

对人的跟踪包括对行为的跟踪和对健康数据的跟踪。行为数据通过模型分析可以推测健康数据。个人对于自我的跟踪目的也是更好的了解自己，改善自我效率、目标、价值、健康等。

跟踪经济的核心是人。第一代通过邮箱或帐号作为识别ID，用cookies跟踪个体的上网行为，投其所好，引导消费。第二代用手机号作为识别ID，手机、电脑多方位收集支付信息、位置信息、交流信息更多维度刻画用户。第三代利用可联网的传感器从外部收集人的信息，各种维度，能更准确的识别人，滴滴出行、共享单车、支付宝。

腾讯通过发身份证，再引导用户使用各种网络服务、社交活动，了解人。阿里通过网络行为、线下团购、出行等特征，合成一个人。最终两家公司会完全了解你，你的性格、喜好、收入、工作，完全掌握，然后把你玩弄于手掌之中。

家庭摄像头产生原因是老龄化、二娃及对安全缺失的恐惧。智能手环收集运动信息并做朋友排名，满足健康和虚荣需求。智能跑鞋和运动手表用于收集和分析运动数据，优化训练。儿童手表跟踪位置和主动确定行为，解决父母对儿童安全的恐惧。
