2018年终总结

发表于 2018-12-29

2018 年最后一个工作日，总结过去，展望未来。

总结 2018

2017 年末总结时的 2018 年计划

财务目标：在 6 月份换工作，对比上半年和下半年，增幅 67.9%。基本达标。
健康目标：没有坚持，没有数据记录，可能断断续续晨跑了 20 天，每天 30 分钟。不达标
家庭目标：达标，只是 2019 年才正式宣告。
个人目标：电子琴购入，但没有练习。老是以缺琴架、乐谱、妹子不想玩为接口。不达标。
其他大事记：换房，通勤时间每天减少 2 小时，身心较为愉悦，但是没有充分利用这花钱买的 2 小时。
学习、阅读、写作：订阅了技术专栏 2 个，泛泛收听没有有效吸收。阅读了不少书，写了 13 篇读书笔记，剩下可能有一半的没写读书笔记。内网写了好多过程文档，博客文章全年写了 57 篇。
python django 入了门，使用 django，集成一个 markdown 编辑器，做了一个简单 wiki。
linux 系统操作熟练了，编译程序、根据日志、google 排查故障 ok，深入原理还没有。

展望 2019

2018 年时间还是溜得悄无声息，需要重读《奇特的一生》，在这段时间旅行上刻画更多的痕迹。
工作的一切，都是为了生活。要想过得有点趣，生活总得带点绿。健康的身体、有益的爱好是必要的。
财务目标：double，通过投资自己，投资资金等方式。
健康目标：跑步，每周一、三、五，除非下大雨影响跑步，都要去，不能因为天气太热、太冷等因素放弃。156 天，带上手环年底看运动总结。
趣味目标：

  • 电子琴增强练习

  • 英语流利说半年特训

  • 阅读好玩的书

  • 与妹子记录娃相关

能力提升目标：

  • redhat 中级证书

  • python 数据分析及自动化运维能力提升

  • 写作能力提升，思考金字塔原理、讲故事、写作技法、排版相关。

  • 演说能力提升。。。

计划没有变化快，唯一不变的就是变化本身，但是我们还是要制定计划。短期目标称为计划，长期目标叫战略。像我的偶像费曼、王小波、富兰克林等致敬，他们是我的灯塔，时常眺望才不至于迷失方向。
