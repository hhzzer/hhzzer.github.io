Django 环境配置

3 安装 django 并验证

根据 django tutorial 熟悉开发环境
app 是一个站点的应用，站点可理解为一个项目。
~/code/python3/dj-project$ python manage.py startapp polls

url 创建页面路由
model 描述数据模型及数据方法
